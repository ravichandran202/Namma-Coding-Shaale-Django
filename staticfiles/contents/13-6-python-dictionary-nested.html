Nested Dictionaries

Learn how to manage complex data structures by storing dictionaries inside other dictionaries.

1. Structure of Nested Dictionaries

A nested dictionary is simply a dictionary where the values are themselves dictionaries. This is commonly used to represent structured data like JSON APIs, databases, or configuration files.

Basic Syntax
# A dictionary of students
# Keys are student IDs, Values are dictionaries of details
class_records = {
    "student_01": {
        "name": "Alice",
        "age": 12,
        "grades": {"math": 90, "science": 85}
    },
    "student_02": {
        "name": "Bob",
        "age": 13,
        "grades": {"math": 75, "science": 92}
    }
}

print(class_records)
 Output
{'student_01': {'name': 'Alice', 'age': 12, 'grades': {'math': 90, 'science': 85}}, 'student_02': {'name': 'Bob', 'age': 13, 'grades': {'math': 75, 'science': 92}}}

2. Accessing Nested Elements

To access data inside a nested dictionary, you chain the square brackets [] or get() methods.

my_family = {
  "child1": {"name": "Emil", "year": 2004},
  "child2": {"name": "Tobias", "year": 2007},
  "child3": {"name": "Linus", "year": 2011}
}

# 1. Access the dictionary for child2
print(my_family["child2"])

# 2. Access specifically the name of child2
name = my_family["child2"]["name"]
print("Child 2 Name:", name)

# 3. Access using .get() (Safer)
year = my_family.get("child3").get("year")
print("Child 3 Year:", year)
 Output
{'name': 'Tobias', 'year': 2007}
Child 2 Name: Tobias
Child 3 Year: 2011
Visualization

Think of it like a file path: dictionary -> key -> sub_key -> value.
my_family["child2"]["name"] means: Go to my_family, find child2, then inside that, find name.

3. Modifying Nested Data

Adding and Updating
products = {
    "laptop": {"price": 1000, "stock": 5},
    "mouse": {"price": 20, "stock": 10}
}

# 1. Update a value inside a nested dictionary
products["laptop"]["price"] = 950  # Sale price!
products["laptop"]["stock"] -= 1   # Sold one

# 2. Add a new key-value pair to a nested dictionary
products["mouse"]["color"] = "black"

# 3. Add a completely new nested item
products["keyboard"] = {"price": 50, "stock": 8}

print(products)
 Output
{
    'laptop': {'price': 950, 'stock': 4}, 
    'mouse': {'price': 20, 'stock': 10, 'color': 'black'}, 
    'keyboard': {'price': 50, 'stock': 8}
}

4. Iterating Nested Dictionaries

Looping through complex data
people = {
    1: {'name': 'John', 'age': '27', 'sex': 'Male'},
    2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}
}

for p_id, p_info in people.items():
    print(f"\nPerson ID: {p_id}")
    
    # p_info is the inner dictionary
    for key in p_info:
        print(f"{key}: {p_info[key]}")
 Output
Person ID: 1
name: John
age: 27
sex: Male

Person ID: 2
name: Marie
age: 22
sex: Female

Try It Yourself

You have a dictionary representing a library system:

library = {
    "Harry Potter": {"author": "J.K. Rowling", "available": True},
    "The Hobbit": {"author": "Tolkien", "available": False}
}

Perform the following operations:

  1. Print the author of "Harry Potter".
  2. Change the availability of "The Hobbit" to True.
  3. Add a new book "1984" with author "Orwell" and available True.
  4. Write a loop to print only the books that are currently available.
library = {
    "Harry Potter": {"author": "J.K. Rowling", "available": True},
    "The Hobbit": {"author": "Tolkien", "available": False}
}

# 1. Print Author
print("1. Author of HP:", library["Harry Potter"]["author"])

# 2. Change Availability
library["The Hobbit"]["available"] = True
print("\n2. Hobbit is now available.")

# 3. Add new book
library["1984"] = {"author": "Orwell", "available": True}
print("\n3. Library updated.")

# 4. Loop for available books
print("\n4. Available books:")
for book, details in library.items():
    if details["available"] == True:
        print(f"- {book} by {details['author']}")
     Output
    1. Author of HP: J.K. Rowling

2. Hobbit is now available.

3. Library updated.

4. Available books:
- Harry Potter by J.K. Rowling
- The Hobbit by Tolkien
- 1984 by Orwell
Nested Dictionary Summary
Operation Syntax Example Description
Access Depth 1 dict["key1"] Returns the inner dictionary
Access Depth 2 dict["key1"]["key2"] Returns the specific value
Modify Nested dict["key1"]["key2"] = val Updates inner value
Add Nested dict["new_key"] = { ... } Adds a new dictionary entry
