Python Tuples

Tuples are ordered, immutable collections that can store multiple items of different types.

  • Ordered: Items have a defined order that won't change
  • Immutable: You cannot modify the tuple after creation
  • Allows Duplicates: Tuples can have items with the same value
  • Heterogeneous: Can contain elements of different data types
Syntax
# With parentheses (recommended)
tuple_name = (item1, item2, ...)

# Without parentheses (valid but less readable)
tuple_name = item1, item2, ...
Example
my_tuple = (10, "Hello", 3.5, True)
print(my_tuple)
 Output
(10, 'Hello', 3.5, True)

1. Creating Tuples

Create tuples using parentheses or just commas

# A tuple of strings
fruits = ("apple", "banana", "cherry")
print("Fruits:", fruits)

# A tuple of integers
prime_numbers = (2, 3, 5, 7, 11)
print("Primes:", prime_numbers)

# A tuple with mixed data types
mixed_bag = (42, "hello", 3.14, True)
print("Mixed:", mixed_bag)

# An empty tuple
empty_tuple = ()
print("Empty:", empty_tuple)

# Single element tuple (requires trailing comma)
single_element = ("hello",)
print("Single:", single_element)

# Tuple without parentheses
coordinates = 10, 20, 30
print("Coordinates:", coordinates)
 Output
Fruits: ('apple', 'banana', 'cherry')
Primes: (2, 3, 5, 7, 11)
Mixed: (42, 'hello', 3.14, True)
Empty: ()
Single: ('hello',)
Coordinates: (10, 20, 30)
Important Note

For single-element tuples, you must include a trailing comma. Without it, Python treats it as a regular value in parentheses.

single_correct = ("hello",)    # This is a tuple
single_wrong = ("hello")      # This is a string!

2. Accessing Elements

Access elements using positive or negative indexing

Positive Indexing
elements = ("Alice", 100, 7.5, 0)
    
# Access the first element (index 0)
first_element = elements[0]
print("First element:", first_element)

# Access the third element (index 2)
third_element = elements[2]
print("Third element:", third_element)
 Output
First element: Alice
Third element: 7.5
Negative Indexing
elements = ("Alice", 100, 7.5, 0)
    
# Access the last element
last_element = elements[-1]
print("Last element:", last_element)

# Access the second-to-last element
second_last_element = elements[-2]
print("Second last:", second_last_element)
 Output
Last element: 0
Second last: 7.5
Tuple Index Visualization
Element "Alice" 100 7.5 0
Positive Index elements[0] elements[1] elements[2] elements[3]
Negative Index elements[-4] elements[-3] elements[-2] elements[-1]

3. Looping Through Tuples

Method 1: Loop by Element

Directly access each element

fruits = ("apple", "banana", "cherry")
    
print("My favorite fruits are:")
for fruit in fruits:
    print(f"- {fruit}")
 Output
My favorite fruits are:
- apple
- banana
- cherry
Method 2: Loop by Index

Access elements using index positions

fruits = ("apple", "banana", "cherry")
    
print("Tuple with indices:")
for i in range(len(fruits)):
    print(f"Index {i}: {fruits[i]}")
 Output
Tuple with indices:
Index 0: apple
Index 1: banana
Index 2: cherry
Method 3: Using enumerate()

Get both index and value simultaneously

fruits = ("apple", "banana", "cherry")
    
for index, value in enumerate(fruits):
    print(f"fruits[{index}] = '{value}'")
 Output
fruits[0] = 'apple'
fruits[1] = 'banana'
fruits[2] = 'cherry'

Practical Examples

# Student record example
student = ("Alice", 85, "Computer Science", 3.8)

print("Student Information:")
print(f"Name: {student[0]}")
print(f"Grade: {student[1]}")
print(f"Major: {student[2]}")
print(f"GPA: {student[3]}")
 Output
Student Information:
Name: Alice
Grade: 85
Major: Computer Science
GPA: 3.8
# RGB color values
colors = [
    ("red", (255, 0, 0)),
    ("green", (0, 255, 0)),
    ("blue", (0, 0, 255))
]

print("Color Codes:")
for color_name, rgb in colors:
    print(f"{color_name}: {rgb}")
 Output
Color Codes:
red: (255, 0, 0)
green: (0, 255, 0)
blue: (0, 0, 255)

Create a tuple with the following data:

Then complete these tasks:

  1. Print the entire tuple
  2. Access and print the second element
  3. Access and print the last element using negative indexing
  4. Loop through and print each element
  5. Loop through and print elements with their indices
# Create the tuple
language_info = ("Python", 1991, "Guido van Rossum", 3.9)

# 1. Print entire tuple
print("1. Entire tuple:", language_info)

# 2. Access second element

# 3. Access last element with negative indexing

# 4. Loop through elements
print("4. All elements:")
for element in language_info:
    print(f"  - {element}")

# 5. Loop with indices
print("5. Elements with indices:")
for i in range(len(language_info)):
    print(f"  Index {i}: {language_info[i]}")
 Output

2. Second element: 1991

3. Last element: 3.9

4. All elements:
  - Python
  - 1991
  - Guido van Rossum
  - 3.9
  
5. Elements with indices:
  Index 0: Python
  Index 1: 1991
  Index 2: Guido van Rossum
  Index 3: 3.9
Remember
  • Tuples are ordered and immutable
  • Use parentheses () to create tuples (optional but recommended)
  • Single-element tuples require a trailing comma: ("hello",)
  • Access elements using [index] with positive or negative indices
  • Use loops to iterate through all elements
  • Tuples are faster than lists for fixed data that won't change
