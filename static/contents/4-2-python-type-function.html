Type() Function

In Python, type() is a built-in function that is mainly used to check the type of an object 

# python type() function syntax
variable = type(value)

Example

1. Numeric Types:

# Numeric Types
x = 10
print(type(x))  

y = 3.14
print(type(y))  

z = 2 + 5j
print(type(z))
 Output
<class 'int'>
<class 'float'>
<class 'complex'>

2. Sequence Types:

# Sequence Types
text = "Hello"
print(type(text))  

my_list = [1, 2, 3]
print(type(my_list))  

my_tuple = (1, 2, 3)
print(type(my_tuple))   

my_range = range(5)
print(type(my_range))
 Output
<class 'str'>
<class 'list'>
<class 'tuple'>
<class 'range'>

3. Mapping Type:

# Mapping Type
person = {'name': 'Alice', 'age': 30}
print(type(person))

 Output
                   <class 'dict'>

4. Set Type:

# Set Types
my_set = {1, 2, 3}
print(type(my_set))

 Output

<class 'set'>

5. Boolean Type:

# example of python bool datatype
bool_var = True

bool_var2 = False

print(type(bool_var))
print(type(bool_var2))


 Output

<class 'bool'>
<class 'bool'>

6. None Type:

# example of python none datatype
none_var = None

print(type(none_var))

 Output

<class 'none'>
