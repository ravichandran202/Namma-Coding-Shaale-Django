Dictionary Operations

Learn how to create, access, modify, and iterate through dictionaries in Python

1. Creating Dictionaries

Method A: Using Curly Braces {}
# Empty dictionary
empty_dict = {}
print("Empty:", empty_dict)
 Output
Empty: {}
# Dictionary with string keys
person = {
    "name": "Alice", 
    "age": 25, 
    "city": "New York"
}
print("Person:", person)
 Output
Person: {'name': 'Alice', 'age': 25, 'city': 'New York'}
# Dictionary with different key types
mixed_keys = {
    "name": "Bob", 
    1: "one", 
    2.5: "two point five"
}
print("Mixed keys:", mixed_keys)

 Output
Mixed keys: {'name': 'Bob', 1: 'one', 2.5: 'two point five'}
# Dictionary with list as value
student = {
    "name": "John", 
    "scores": [85, 92, 78]
}
print("Student:", student)
 Output
Student: {'name': 'John', 'scores': [85, 92, 78]}
Method B: Using dict() Constructor
# Using keyword arguments
person = dict(name="Alice", age=25, city="Boston")
print("Using keywords:", person)

# Using list of tuples
fruits = dict([("apple", 5), ("banana", 3), ("orange", 2)])
print("Using tuples:", fruits)

# From two lists (using zip)
keys = ["name", "age", "city"]
values = ["Bob", 30, "Chicago"]
person2 = dict(zip(keys, values))
print("From two lists:", person2)
 Output
Using keywords: {'name': 'Alice', 'age': 25, 'city': 'Boston'}
Using tuples: {'apple': 5, 'banana': 3, 'orange': 2}
From two lists: {'name': 'Bob', 'age': 30, 'city': 'Chicago'}

2. Accessing Elements

student = {
   "name": "John", 
   "age": 20, 
   "grade": "A"
}

# Access using key
name = student["name"]
age = student["age"]
print("Name:", name)
print("Age:", age)

# Access using get() method (safer)
grade = student.get("grade")
city = student.get("city", "Not specified")  # Default value
print("Grade:", grade)
print("City:", city)
 Output
Name: John
Age: 20
Grade: A
City: Not specified
Important Note

Use get() method when you're not sure if a key exists. It returns None (or a default value) instead of raising a KeyError.

3. Adding and Modifying Elements

Adding New Elements
student = {"name": "John", "age": 20}
print("Original:", student)

# Add new key-value pairs
student["grade"] = "A"
student["city"] = "Boston"

print("After adding:", student)

# Add multiple elements using update()
student.update({"country": "USA", "major": "Computer Science"})
print("After update:", student)
 Output
Original: {'name': 'John', 'age': 20}
After adding: {'name': 'John', 'age': 20, 'grade': 'A', 'city': 'Boston'}
After update: {'name': 'John', 'age': 20, 'grade': 'A', 'city': 'Boston', 'country': 'USA', 'major': 'Computer Science'}
Modifying Existing Elements
student = {"name": "John", "age": 20, "grade": "B"}
print("Original:", student)

# Modify existing values
student["grade"] = "A"  # Changes grade from B to A
student["age"] = 21     # Changes age from 20 to 21

print("After modification:", student)

# Modify using update()
student.update({"age": 22, "city": "New York"})
print("After update:", student)
 Output
Original: {'name': 'John', 'age': 20, 'grade': 'B'}
After modification: {'name': 'John', 'age': 21, 'grade': 'A'}
After update: {'name': 'John', 'age': 22, 'grade': 'A', 'city': 'New York'}

4. Iterating Through Dictionaries

Loop Through Keys
student = {"name": "John", "age": 20, "grade": "A"}
    
print("Method 1: Direct looping (gets keys)")
for key in student:
    print(f"Key: {key}")

print("\nMethod 2: Using keys() method")
for key in student.keys():
    print(f"Key: {key}")
 Output
Method 1: Direct looping (gets keys)
Key: name
Key: age
Key: grade

Method 2: Using keys() method
Key: name
Key: age
Key: grade
Loop Through Values
student = {"name": "John", "age": 20, "grade": "A"}
    
print("Loop through values:")
for value in student.values():
    print(f"Value: {value}")

# Practical example: calculate average score
scores_dict = {"math": 85, "science": 92, "english": 78}
total = 0
count = 0

for score in scores_dict.values():
    total += score
    count += 1

average = total / count
print(f"\nAverage score: {average:.1f}")
 Output
Loop through values:
Value: John
Value: 20
Value: A

Average score: 85.0
Loop Through Key-Value Pairs
student = {"name": "John", "age": 20, "grade": "A"}
    
print("Loop through key-value pairs:")
for key, value in student.items():
    print(f"{key}: {value}")

# Practical example: phone book
phone_book = {
    "Alice": "9876543210",
    "Bob": "8765432109",
    "Charlie": "7654321098"
}

print("\nPhone Book:")
for name, number in phone_book.items():
    print(f"{name}: {number}")
 Output
Loop through key-value pairs:
name: John
age: 20
grade: A

Phone Book:
Alice: 9876543210
Bob: 8765432109
Charlie: 7654321098

Dictionary Operations Summary
Operation Syntax Description
Create {key: value} or dict() Create new dictionary
Access dict[key] or dict.get(key) Get value by key
Add dict[new_key] = value Add new key-value pair
Modify dict[key] = new_value Update existing value
Loop Keys for key in dict: Iterate through keys
Loop Values for value in dict.values(): Iterate through values
Loop Both for k,v in dict.items(): Iterate through key-value pairs
