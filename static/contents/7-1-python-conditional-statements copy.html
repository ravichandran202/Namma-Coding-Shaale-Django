Conditional Statements

Conditional statements allow your program to make decisions and execute different code blocks based on whether conditions are met.

They evaluate expressions to True or False and execute code accordingly.

Types of conditional statements
  • if Statement
  • if-else Statement
  • if-elif-else Statement
  • Nested Conditional Statements
  • match-case Statement
  • Ternary Operator

  1. if Statement

    The basic conditional statement that executes code only if a condition is True. 

    # Syntax
if condition:
    # code to run if condition is True
    # Example: Check if a number is positive
number = 5
if number > 0:
    print("This number is positive!") 
 Output
This number is positive!
    Step-by-Step Evaluation
    1. Variable number is assigned value 5
    2. Condition number > 0 is evaluated: 5 > 0 → True
    3. Since condition is True, the code block executes
    4. Prints: "This number is positive!"
    if Statement Flow
    graph TD; A[Start] --> B{condition}; B -->|True| C[Execute if block]; B -->|False| D[Skip if block]; C --> E[Continue program]; D --> E;
    if statement flowchart: Code executes only when condition is True

  2. if-else Statement

    Executes one block of code if the condition is True, and another if it's False. 

    # Syntax
if condition:
    # code if condition is True
else:
    # code if condition is False
    # Example: Check if number is even or odd
number = 7
if number % 2 == 0:
    print("Even number!")
else:
    print("Odd number!") 
 Output
Odd number!
    Step-by-Step Evaluation
    1. Variable number is assigned value 7
    2. Condition number % 2 == 0 is evaluated: 7 % 2 = 1 → 1 == 0 → False
    3. Since condition is False, the else block executes
    4. Prints: "Odd number!"
    if-else Statement Flow
    graph TD; A[Start] --> B{condition}; B -->|True| C[Execute if block]; B -->|False| D[Execute else block]; C --> E[Continue program]; D --> E;
    if-else statement flowchart: Always executes one of the two blocks

  3. if-elif-else Statement

    This statement is used to check multiple conditions in sequence.

    # Syntax
if condition1:
    # code if condition1 is True
elif condition2:
    # code if condition2 is True
else:
    # code if all conditions are False
    # Example: Greeting based on time
hour = 14  # 2 PM
if hour < 12:
    print("Good morning!")
elif hour < 18:
    print("Good afternoon!")
else:
    print("Good evening!") 
 Output
Good afternoon!
    Step-by-Step Evaluation
    1. Variable hour is assigned value 14
    2. First condition hour < 12: 14 < 12 → False
    3. Second condition hour < 18: 14 < 18 → True
    4. Since second condition is True, the elif block executes
    5. Prints: "Good afternoon!"
    if-elif-else Statement Flow
    graph TD; A[Start] --> B{condition1}; B -->|True| C[Execute if block]; B -->|False| D{condition2}; D -->|True| E[Execute elif block]; D -->|False| F[Execute else block]; C --> G[Continue program]; E --> G; F --> G;
    if-elif-else flowchart: Checks conditions in sequence until one is True

  4. Nested Conditionals

    Conditional statements inside another conditional statement.

    # Syntax
if condition1:
    if condition2:
        # code if both conditions are True
    # Example: Check number type
number = 10
if number > 0:
    if number % 2 == 0:
        print("Positive even number")
    else:
        print("Positive odd number")
else:
    print("Number is not positive") 
 Output
Positive even number
    Step-by-Step Evaluation
    1. Variable number is assigned value 10
    2. First condition number > 0: 10 > 0 → True
    3. Enters the outer if block
    4. Second condition number % 2 == 0: 10 % 2 = 0 → 0 == 0 → True
    5. Enters the inner if block
    6. Prints: "Positive even number"
    Nested Conditionals Flow
    graph TD; A[Start] --> B{Outer condition}; B -->|True| C{Inner condition}; B -->|False| D[Execute outer else]; C -->|True| E[Execute inner if]; C -->|False| F[Execute inner else]; D --> G[Continue program]; E --> G; F --> G;
    Nested conditionals flowchart: Conditions within conditions for complex logic

  5. Match Case Statement

    The match statement is used to perform different actions based on different conditions (Python 3.10+).

    # Syntax
match subject:
    case pattern1:
        # code if pattern1 matches
    case pattern2:
        # code if pattern2 matches
    case _:
        # default case if no other matches
    # Example: Day of the week
day = 4
match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case 4:
        print("Thursday")
    case 5:
        print("Friday")
    case 6:
        print("Saturday")
    case 7:
        print("Sunday")
    case _:
        print("Invalid Day") 
 Output
Thursday
    Step-by-Step Evaluation
    1. Variable day is assigned value 4
    2. match statement compares day with each case
    3. Matches case 4: pattern
    4. Executes the code block for case 4:
    5. Prints: "Thursday"
    Match Case Statement Flow
    graph TD; A[Start] --> B[Match subject]; B --> C{Case pattern1}; C -->|Match| D[Execute case1]; C -->|No match| E{Case pattern2}; E -->|Match| F[Execute case2]; E -->|No match| G{Case patternN}; G -->|Match| H[Execute caseN]; G -->|No match| I[Execute default]; D --> J[Continue program]; F --> J; H --> J; I --> J;
    match-case flowchart: Pattern matching with multiple possible outcomes

  6. Ternary Operator

    A shorthand way to write simple if-else statements.

    # Syntax
variable = value_if_true if condition else value_if_false
    # Example: Check even or odd
number = 4
result = "Even" if number % 2 == 0 else "Odd"
print(result) 
 Output
Even
    Step-by-Step Evaluation
    1. Variable number is assigned value 4
    2. Condition number % 2 == 0 is evaluated: 4 % 2 = 0 → 0 == 0 → True
    3. Since condition is True, value_if_true ("Even") is assigned
    4. Variable result becomes "Even"
    5. Prints: "Even"
Conditional Statements Comparison
Statement Type Use Case Best For
if Single condition check When you only need to check one condition
if-else Two possible outcomes When there are exactly two possible results
if-elif-else Multiple conditions When you have more than two possible scenarios
Nested Complex conditions When conditions depend on other conditions
match-case Pattern matching When comparing a value against multiple patterns
Ternary Simple assignments Short, simple if-else assignments in one line
Practice Exercise

Write a Python program that:

  1. Takes a student's score as input (0-100)
  2. Prints the grade based on:
    • A: 90-100
    • B: 80-89
    • C: 70-79
    • D: 60-69
    • F: Below 60

Try to implement this using different conditional statements.

# Solution using if-elif-else
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F") 
 Output
Grade: B
Step-by-Step Evaluation (for score = 85)
  1. First condition score >= 90: 85 >= 90 → False
  2. Second condition score >= 80: 85 >= 80 → True
  3. Executes the elif score >= 80: block
  4. Prints: "Grade: B"
