Nested Loops

Nested loops occur when one loop is placed inside another loop.

They are useful for working with multi-dimensional data structures, generating patterns, and processing combinations of elements.

The will be executed one time for each iteration of the Outer Loop.

How Nested Loops Work

Important Rule: The inner loop completes ALL its cycles for EACH cycle of the outer loop.

Think of it like: Outer loop = rows, Inner loop = columns

Basic Syntax
# Simple structure
for outer in outer_sequence:
    # Outer loop code
    for inner in inner_sequence:
        # Inner loop code
        # Can use both outer and inner
    # Back to outer loop
Example: Multiplication Table
# Using for loops
print("Multiplication Table (1-2):")
for i in range(1, 3):     # Outer: 1, 2
    for j in range(1, 11): # Inner: 1 to 10
        print(f"{i} x {j} = {i * j}")
    print()  # Blank line between tables

How It Works
  1. Outer loop: i = 1
  2. Inner loop runs 10 times (j = 1 to 10)
  3. Prints 1×1, 1×2, ..., 1×10
  4. Outer loop: i = 2
  5. Inner loop runs 10 times again
  6. Prints 2×1, 2×2, ..., 2×10
Same Example with While Loops
# Using while loops
print("Multiplication Table (while loops):")
i = 1
while i < 3:           # Outer: i = 1, 2
    j = 1
    while j < 11:      # Inner: j = 1 to 10
        print(f"{i} x {j} = {i * j}")
        j += 1
    print()  # Blank line
    i += 1

Example: Square Pattern
# Print a Rectangle pattern
print("Rectangle Pattern:")
for i in range(1, 6):        # Rows: 1 to 5
    for j in range(5):        # Stars: 0 to 4
        print("* ", end="")
    print()  # New line after each row 
 Output

* * * * * 
* * * * * 
* * * * * 
* * * * *
Pattern Logic
  • Row 1: 5 star
  • Row 2: 5 stars
  • Row 3: 5 stars
  • Row 4: 5 stars
  • Row 5: 5 stars
Quick Facts
Loop Type When to Use
Nested for When you know how many iterations
Nested while When iterations depend on conditions
Mixed for inside while or vice versa
