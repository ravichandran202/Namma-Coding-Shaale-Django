Python Functions

Functions are organized blocks of code that are defined once and can be used (called) multiple times.

1. Syntax & Rules

A. Defining a Function

To create a function, we use the def keyword.

# Syntax
def function_name():
    # block of code
    # to be executed
B. Calling a Function

To use the function, we write its name followed by parentheses.

# Syntax
function_name()
Rules for Declaring Functions
  • Keyword: Must start with def.
  • Naming: Use lowercase letters and underscores (e.g., my_function_name). Do not start with numbers or use spaces.
  • Parentheses: The name must be followed by ().
  • Colon: The line must end with a colon :.
  • Indentation: The code inside the function must be indented (usually 4 spaces).

2. Types of Functions

In Python, functions are broadly categorized into two types based on who created them.

A. User-Defined Functions

These are functions created by you (the programmer) to perform specific tasks relevant to your project.

# 1. We DEFINE the function
def say_hello():
    print("Hello from a User-Defined Function!")

# 2. We CALL the function
say_hello()
 Output
Hello from a User-Defined Function!
B. Built-in Functions

These are pre-written functions provided by Python. You don't need to define them; you just call them.

Function Description
print() Displays output to the console.
len() Returns the length of a list or string.
type() Returns the data type of a value.
input() Accepts user input from the console. 
# Using Built-in functions
text = "Python"

# len() is built-in
length = len(text) 

# print() is built-in
print(f"The length is {length}")

3. Basic Concepts

A. Passing Data (Arguments)

We can send information into a function so it can process it. These are called parameters or arguments.

def greet_user(name):
    print(f"Welcome, {name}!")

# Passing "Alice" into the function
greet_user("Alice")
greet_user("Bob")
B. Getting Data Back (Return)

A function can send a result back to the code that called it using the return keyword.

def double_number(num):
    return num * 2

result = double_number(5)
print(result) # Prints 10

4. Try It Yourself

Practice Tasks: Write python code for the following.

  1. Create a function named welcome that prints "Welcome to Python!".
  2. Create a function named square that takes one number as an argument and returns its square (number * number).
  3. Create a function named check_age that takes an age. If age is 18 or more, print "Adult", otherwise print "Child".
Solutions:
# 1. Welcome Function
def welcome():
    print("Welcome to Python!")

# 2. Square Function (Returning a value)
def square(num):
    return num * num

# 3. Check Age Function (Using logic)
def check_age(age):
    if age >= 18:
        print("Adult")
    else:
        print("Child")

# Testing the functions
welcome()
print(square(5))  # Should print 25
check_age(20)     # Should print Adult
Summary
  • Define using def, Call using parentheses ().
  • Built-in functions (like print) come with Python.
  • User-defined functions are custom made by you.
  • Indent your code inside the function definition.
