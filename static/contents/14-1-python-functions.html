Python Functions

Functions are reusable blocks of code that perform specific tasks - making your code organized and efficient!

What is a Function?
  • Reusable block of code that runs when called
  • Accepts data as parameters
  • Returns data as result
  • Organizes code and avoids repetition

1. Basic Function

Syntax

function will be declared via def keywork

# Function declaration
def function_name():
    # block of code

function_name() #calling a function
Example
# Function declaration
def greet():
    print("Good Morning!")

# Calling the function
greet()
 Output
Good Morning!

2. Function with Arguments

We can pass values to a function using an Argument.

Syntax
# Function with n parameters
def function_name(param1, param2, param3, ..... paramn):
    # block of code

function_name(arg1, arg2, arg3, .... argn) #calling a function
Example
# Function with one parameter
def greet_name(name):
    print(f"Hello, {name}!")

# Calling with different arguments
greet_name("Namma Coding Shaale")
greet_name("Ram")
greet_name("Sita")
 Output
Hello, Namma Coding Shaale!
Hello, Ram!
Hello, Sita!
# Function with multiple parameters
def introduce(name, age, city):
    print(f"My name is {name}, I'm {age} years old, and I live in {city}.")

# Calling the function
introduce("Alice", 25, "Bangalore")
introduce("Bob", 30, "Mysore")
 Output
My name is Alice, I'm 25 years old, and I live in Bangalore.
My name is Bob, I'm 30 years old, and I live in Mysore.

3. Returning Values

To return a value from a function, use the return keyword. The function will then give back the final result to the place where it was called.

# Function that returns a value
def function_name(param1, ....paramn):
    #block of code
    return return_value
 
return_value = function_name(arg1, ....argn)
print(return_value)
# Function that returns a value
def add(a, b):
    return a + b   # returns the sum

# Using the returned value
result = add(5, 3)
print("5 + 3 =", result)
 Output
5 + 3 = 8
# More return examples
def is_even(number):
    return number % 2 == 0

number = 10
if is_even(number):
    print(f"{number} is even")
else:
    print(f"{number} is odd")
 Output
10 is even

4. Built-in Functions

We are already using functions that are predefined in Python. Built-in functions are readily available for reuse

# We've been using built-in functions all along!
    
# print() - displays output
print("Hello World!")

# len() - returns length
name = "Python"
print(f"Length of '{name}':", len(name))

# int(), str(), float() - type conversion
number_str = "123"
number_int = int(number_str)
print(f"String '{number_str}' to integer:", number_int)

# max(), min() - find maximum and minimum
numbers = [5, 2, 8, 1, 9]
print("Numbers:", numbers)
print("Maximum:", max(numbers))
print("Minimum:", min(numbers))

# sum() - add all elements
print("Sum:", sum(numbers))
 Output
Hello World!
Length of 'Python': 6
String '123' to integer: 123
Numbers: [5, 2, 8, 1, 9]
Maximum: 9
Minimum: 1
Sum: 25

Practical Examples

# Calculator functions
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b != 0:
        return a / b
    else:
        return "Error: Cannot divide by zero!"

# Using the calculator
print("Calculator Examples:")
print("10 + 5 =", add(10, 5))
print("10 - 5 =", subtract(10, 5))
print("10 × 5 =", multiply(10, 5))
print("10 ÷ 5 =", divide(10, 5))
print("10 ÷ 0 =", divide(10, 0))
 Output
Calculator Examples:
10 + 5 = 15
10 - 5 = 5
10 × 5 = 50
10 ÷ 5 = 2.0
10 ÷ 0 = Error: Cannot divide by zero!

Try It Yourself

Create the following functions:

  1. A function that takes a name and returns a greeting message
  2. A function that calculates the area of a rectangle (length × width)
  3. A function that checks if a number is positive, negative, or zero
  4. A function that takes a list of numbers and returns their average
  5. A function that converts Celsius to Fahrenheit (F = C × 9/5 + 32)
# 1. Greeting function
def greet_person(name):
    return f"Hello, {name}! Welcome to Python programming."

# 2. Area of rectangle
def rectangle_area(length, width):
    return length * width

# 3. Number checker
def check_number(num):
    if num > 0:
        return "Positive"
    elif num < 0:
        return "Negative"
    else:
        return "Zero"

# 4. Average calculator
def calculate_average(numbers):
    if len(numbers) == 0:
        return 0
    return sum(numbers) / len(numbers)

# 5. Temperature converter
def celsius_to_fahrenheit(celsius):
    return celsius * 9/5 + 32

# Testing the functions
print("1. Greeting:", greet_person("Namma Coding Shaale"))

print("\n2. Rectangle area:")
print("Area of 5×3 rectangle:", rectangle_area(5, 3))

print("\n3. Number check:")
print("10 is:", check_number(10))
print("-5 is:", check_number(-5))
print("0 is:", check_number(0))

print("\n4. Average calculation:")
scores = [85, 92, 78, 96, 88]
print("Scores:", scores)
print("Average:", calculate_average(scores))

print("\n5. Temperature conversion:")
print("0°C =", celsius_to_fahrenheit(0), "°F")
print("100°C =", celsius_to_fahrenheit(100), "°F")
print("25°C =", celsius_to_fahrenheit(25), "°F")
 Output
1. Greeting: Hello, Namma Coding Shaale! Welcome to Python programming.

2. Rectangle area:
Area of 5×3 rectangle: 15

3. Number check:
10 is: Positive
-5 is: Negative
0 is: Zero

4. Average calculation:
Scores: [85, 92, 78, 96, 88]
Average: 87.8

5. Temperature conversion:
0°C = 32.0 °F
100°C = 212.0 °F
25°C = 77.0 °F
