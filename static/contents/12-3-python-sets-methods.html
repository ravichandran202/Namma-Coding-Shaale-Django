Python Set Methods with Symbols

A set in Python has built-in methods that let you perform operations like union, intersection, difference, and modification.

1. Adding Elements

add() - Add Single Element

Adds a single element to the set. If the element already exists, it does nothing.

s = {1, 2, 3}
s.add(4)
print(s)  # {1, 2, 3, 4}
s.add(2)  
print(s)  # {1, 2, 3, 4} (no duplicates)
 Output
{1, 2, 3, 4}
{1, 2, 3, 4}
update() - Add Multiple Elements

Adds multiple elements from another set, list, tuple, or any iterable.

s = {1, 2}
s.update([3, 4], {5, 6})
print(s)
 Output
{1, 2, 3, 4, 5, 6}

2. Removing Elements

remove() - Remove Element (Raises Error)

Removes the specified element. If the element doesn’t exist → raises KeyError. 

s = {1, 2, 3}
s.remove(2)
print(s)
# s.remove(5)  # This would raise KeyError
 Output
{1, 3}
discard() - Remove Element (No Error)

Removes the specified element. If the element doesn’t exist → no error.

s = {1, 2, 3}
s.discard(2)
print(s)

s.discard(5)  # No error
print(s)
 Output
{1, 3}
{1, 3}
pop() - Remove Random Element

Removes and returns a random element (since sets are unordered).

s = {10, 20, 30}
print(s.pop())
print(s)
 Output
10
{20, 30}
clear() - Remove All Elements

Removes all elements from the set.

s = {1, 2, 3}
s.clear()
print(s)
 Output
set()

3. Copy Method

copy() - Copy All Elements

Returns a shallow copy of the set. ( assign set without reference )

s = {1, 2, 3}
new_s = s.copy()
print(new_s)
 Output
{1, 2, 3}

4. Set Operations with Symbols

union() - Combine Sets (A ∪ B)

Returns a set containing all elements from both sets (duplicates removed).

a = {1, 2}
b = {2, 3, 4}
print("Method:", a.union(b))
print("Symbol:", a | b)
 Output
Method: {1, 2, 3, 4}
Symbol: {1, 2, 3, 4}
intersection() - Common Elements (A ∩ B)

Returns elements common to both sets.

a = {1, 2, 3}
b = {2, 3, 4}
print("Method:", a.intersection(b))
print("Symbol:", a & b)
 Output
Method: {2, 3}
Symbol: {2, 3}
difference() - Elements in First Only (A - B)

Returns elements that are only in the first set, not in others.

a = {1, 2, 3, 4}
b = {3, 4, 5}
print("Method:", a.difference(b))
print("Symbol:", a - b)
 Output
Method: {1, 2}
Symbol: {1, 2}
symmetric_difference() - Elements in Either But Not Both (A ^ B)

Returns elements that are only in the first set, not in others.

a = {1, 2, 3}
b = {3, 4, 5}
print("Method:", a.symmetric_difference(b))
print("Symbol:", a ^ b)
 Output
Method: {1, 2, 4, 5}
Symbol: {1, 2, 4, 5}

5. Set Relationships

issubset() - Check if Subset (A ⊆ B)

Checks if all elements of one set exist in another.

a = {1, 2}
b = {1, 2, 3}
print("Method:", a.issubset(b))
print("Symbol:", a <= b)
 Output
Method: True
Symbol: True
issuperset() - Check if Superset (A ⊇ B)

Checks if a set contains all elements of another set.

a = {1, 2, 3, 4}
b = {2, 3}
print("Method:", a.issuperset(b))
print("Symbol:", a >= b)
 Output
Method: True
Symbol: True
isdisjoint() - Check if No Common Elements

Returns True if two sets have no common elements.

a = {1, 2}
b = {3, 4}
print("Method:", a.isdisjoint(b))
 Output
Method: True
Set Methods Summary with Symbols
Method Symbol Syntax Description
add() - set.add(element) Adds single element
update() - set.update(iterable) Adds multiple elements
remove() - set.remove(element) Removes element (raises error)
discard() - set.discard(element) Removes element (no error)
pop() - set.pop() Removes random element
clear() - set.clear() Removes all elements
copy() - set.copy() Creates shallow copy
union() | set1.union(set2) Combines sets (A ∪ B)
intersection() & set1.intersection(set2) Common elements (A ∩ B)
difference() - set1.difference(set2) Elements in first only (A - B)
symmetric_difference() ^ set1.symmetric_difference(set2) Elements in either but not both (A Δ B)
issubset() <= set1.issubset(set2) Checks if subset (A ⊆ B)
issuperset() >= set1.issuperset(set2) Checks if superset (A ⊇ B)
isdisjoint() - set1.isdisjoint(set2) Checks if no common elements

Practice Exercise

Given the sets:

A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}
C = {5, 6, 9, 10}

Perform these operations using both methods and symbols:

  1. Add 11 to set A
  2. Remove 3 from set A
  3. Find common elements between A and B (A ∩ B)
  4. Find elements in A but not in B (A - B)
  5. Combine all three sets (A ∪ B ∪ C)
  6. Check if B is a subset of A (B ⊆ A)
  7. Find elements that are in either A or B but not both (A Δ B)
  8. Create a copy of set A
