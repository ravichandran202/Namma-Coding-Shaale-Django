Python Modules, Packages, and pip

Learn how to organize and reuse code with modules, packages, and manage external libraries with pip.

1. What is a Module?

A module is simply a Python file (.py) that contains code — functions, variables, or classes — which you can reuse in other programs.

It helps organize large programs into smaller, manageable pieces.

Example Module
# file: math_operations.py
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

PI = 3.14159
Using the Module
# file: main.py
import math_operations

result = math_operations.add(5, 3)
print(result) 
 Output
8

Three Ways to Import Modules

1. import module_name
import math

print(math.sqrt(16))   # Using sqrt() from math module
print(math.pi)         # Accessing pi constant 
 Output
4.0
3.141592653589793

You imported the whole math module, so you use math. before the function name.

2. from module_name import function_name
from math import sqrt, pi

print(sqrt(25))   # No need to write math.sqrt
print(pi)         # Direct access to pi 
 Output
5.0
3.141592653589793

You imported only specific functions from the math module, so you can call them directly.

3. import module_name as alias_name
import math as m

print(m.pi)           # Accessing pi using the alias
print(m.pow(2, 3))    # Using power function 
 Output
3.141592653589793
8.0

Here, math is imported as m, so you can use m instead of math.

2. What is a Package?

A package is a collection of modules arranged in folders (directories).

It must contain a special file named __init__.py (can be empty). This file tells Python that the folder is a package.

Package Structure
my_package/
    __init__.py
    math_operations.py
    string_operations.py
    utils/
        __init__.py
        helpers.py
Using a Package
from my_package import math_operations
from my_package.utils import helpers

result = math_operations.add(10, 5)
print(result) 
 Output
15
Key Point

Packages make it easier to organize related modules. You can create nested packages inside other packages.

3. What is pip?

pip stands for "Python Package Installer".

It is a tool used to install, update, and manage external (third-party) Python packages that are not part of the standard library.

Basic Syntax
# Install a package
pip install package_name

# Example
pip install numpy

This command installs the NumPy library so you can use it in your Python programs.

Basic pip Commands
Command Description
pip install package_name Install a package
pip uninstall package_name Remove a package
pip list List installed packages
pip show package_name Show details about a package
pip install -r requirements.txt Install all packages listed in a file
pip freeze > requirements.txt Save installed packages to a file
Key Takeaways
  • Module: Single Python file with reusable code
  • Package: Collection of modules in a directory with __init__.py
  • pip: Package manager for installing external libraries
  • Import methods: import module, from module import function, import module as alias
  • Use modules to organize code within a project
  • Use packages to organize related modules
  • Use pip to install third-party libraries from PyPI
