Membership Operators

Membership operators (in and not in) are used to check if a value exists within a sequence (like strings, lists, tuples, dictionaries, or sets).

Operator Name Syntax Example Result
in Membership Check x in y 3 in [1, 2, 3] True
not in Negated Check x not in y 4 not in [1, 2, 3] True
  1. Membership Operator (in)

    This operator returns True if the specified value is present in a sequence.

    Syntax: x in sequence

    # example for membership operator with list
x = [1, 2, 3, 4]
print(3 in x)   # Output: True

# example with string
x = "ravi"
print("r" in x) # Output: True

# case-sensitive check
x = "ravi"
print("R" in x) # Output: False

 Output
                        True
True
False
  2. Not Membership Operator (not in)

    This operator returns True if the specified value is not in a sequence.

    Syntax: x not in sequence

    # example for not membership operator with list
x = [1, 2, 3, 4]
print(3 not in x)   # Output: False

# example with string
x = "ravi"
print("r" not in x) # Output: False

# case-sensitive check
x = "ravi"
print("R" not in x) # Output: True

 Output
                        False
False
True
More Examples with Different Data Types 
# With tuples
numbers = (1, 2, 3, 4, 5)
print(3 in numbers)     # True
print(6 not in numbers) # True

# With dictionaries (checks keys)
person = {'name': 'Ravi', 'age': 25}
print('name' in person)     # True
print('city' not in person) # True

# With sets
colors = {'red', 'green', 'blue'}
print('green' in colors)    # True
print('yellow' in colors)   # False

# With strings (substring check)
text = "Hello World"
print("Hello" in text)      # True
print("hello" in text)      # False (case-sensitive)
Operator Comparison

== checks values (content equality)

is checks identity (memory location equality)

in checks membership (value exists in a container)

# Comparison of different operators
x = [1, 2, 3]
y = [1, 2, 3]
z = x

print(x == y)   # True  (same values)
print(x is y)   # False (different objects)
print(x is z)   # True  (same object)
print(2 in x)   # True  (value exists in list)
print(4 in x)   # False (value doesn't exist)
