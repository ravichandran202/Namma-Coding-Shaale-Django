function factorial(n) {
    if (n === 0) return 1;

// Java solution
public class Factorial {
    public static int factorial(int n) {
        if (n == 0) return 1;
        return n * factorial(n - 1);
    }
    
    public static void main(String[] args) {
        System.out.println(factorial(5));
    }
}

Notes & Tips

Tip: Writing Clean Code

Always use meaningful variable names and include comments to explain complex logic. Follow the style guide for the language you're using (PEP 8 for Python). Consistent indentation and proper spacing make your code more readable.

Warning: Common Pitfalls

Avoid using global variables when possible. They can make your code harder to debug and maintain. Also be cautious with mutable default arguments in Python functions as they can lead to unexpected behavior.

Important: Key Concept

Understanding data structures is fundamental to writing efficient programs. Spend time mastering lists, dictionaries, and sets in Python. Know when to use each type for optimal performance.

Danger: Security Risks

Never use eval() with untrusted input as it can execute arbitrary code. Always sanitize user input and use prepared statements for database queries to prevent SQL injection attacks.

1. What does the 'print()' function do in Python?

Correct! The print() function outputs text to the standard output (console).
Incorrect. The print() function outputs to the console, not to a physical printer.

2. Which of these are Python data types? (Select all that apply)

Correct! List, Dictionary, and Tuple are all Python data types. While arrays exist (via the array module), they're not fundamental built-in types.

3. What is the output of the following code?

Correct! Both x and y reference the same list, so modifying y also affects x.
Incorrect. Remember that in Python, lists are mutable and assignment creates a reference, not a copy.

Practice Exercise

Problem: Temperature Converter

Write a Python program that converts temperatures between Fahrenheit and Celsius.

Your program should:

  1. Ask the user to enter a temperature
  2. Ask whether the input is in Fahrenheit (F) or Celsius (C)
  3. Convert to the other scale and display the result
  4. Handle invalid inputs gracefully

The conversion formulas are:

  • Celsius to Fahrenheit: F = (C × 9/5) + 32
  • Fahrenheit to Celsius: C = (F - 32) × 5/9

Solution:

# Temperature Converter Solution
def temp_converter():
    try:
        temp = float(input("Enter temperature: "))
        unit = input("Is this in (F)ahrenheit or (C)elsius? ").upper()
        
        if unit == "F":
            celsius = (temp - 32) * 5/9
            print(f"{temp}°F is {celsius:.1f}°C")
        elif unit == "C":
            fahrenheit = (temp * 9/5) + 32
            print(f"{temp}°C is {fahrenheit:.1f}°F")
        else:
            print("Invalid unit. Please enter F or C.")
    except ValueError:
        print("Invalid temperature. Please enter a number.")

temp_converter()

Note: This solution includes error handling for invalid inputs and formats the output to one decimal place for better readability.
