Python print() Function

Beginner

The print() function is your primary tool for seeing results. It sends data to the standard output device (your screen).

Key Concept
In Python 3, print() is a function, meaning it requires parentheses (). In older Python 2 code, it was a statement (no parentheses needed).

Syntax & Parameters

While you usually just print variables, understanding the optional parameters gives you full control over formatting.

print(*objects, sep=' ', end='\n')
Parameter Default Description
*objects None Any number of objects (strings, numbers, lists) to be printed.
sep ' ' (Space) The string inserted between items if you print more than one.
end '\n' (Newline) The string appended after the last item is printed.

Practical Examples

1. Basic Output

Printing simple strings or numbers.

print("Hello, World!")
print(100)
 Output
Hello, World!
100

2. Using Separators (sep)

Change how items are joined together. Useful for formatting dates or CSV data.

# Default separator is a space
print("Apple", "Banana", "Cherry")

# Custom separator
print("Apple", "Banana", "Cherry", sep=" -> ")
 Output
Apple Banana Cherry
Apple -> Banana -> Cherry

3. Controlling Line Breaks (end)

By default, Python goes to a new line after printing. You can stop this by setting end="".

print("Loading", end="...")
print("Done!")
 Output
Loading...Done!

Knowledge Check

What will be the output of the following code?

print("A", "B", sep="#", end="!")
Correct! The separator # goes between A and B, and ! is at the end.
Incorrect. Remember: sep is between items, end is at the very end.

Python input() Function

Beginner

While print() sends data out to the screen, the input() function gets data in from the user. It allows your program to pause and wait for the user to type something.

The Golden Rule of Input
The input() function ALWAYS returns a string (text), even if the user types a number. If you need to do math, you must convert it first!

Syntax & Parameters

variable_name = input(prompt)
Parameter Description
prompt (Optional) A string that is displayed to the user before they type. It tells them what you want them to enter.

The "Prompt" Parameter: Why use it?

The text inside the parentheses input("...") is called the prompt. It is optional, but without it, your program runs "silently."

Without Prompt

The user sees a blank screen and doesn't know what to do.

# Confusing for the user
name = input() 
print("Hi " + name)
| (Blinking cursor only)

Result: The program waits, but looks "frozen".

With Prompt

The user gets clear instructions on what to type.

# Clear instructions
name = input("Enter name: ")
print("Hi " + name)
Enter name: |

Result: User knows exactly what to type.

Pro Tip: The "Space" Trick
Notice how we added a space at the end of the string: "Enter name: ".
Without that space, the user's typing will be stuck directly to the text (e.g., Enter name:Arjun), which looks messy. Always add a space or a colon!

Practical Examples

1. Basic Text Input

Asking the user for their name and storing it in a variable.

name = input("What is your name? ")
        print("Hello", name)
 User Types: Arjun
 Output: Hello Arjun

2. Taking Number Inputs (Type Casting)

Since input is always a string, we wrap it inside int() or float() to use it as a number.

# incorrect way (causes error or string concatenation)
age = input("Enter your age: ") 

# CORRECT way
age = int(input("Enter your age: "))
next_year = age + 1

print("Next year you will be", next_year)

Knowledge Check

What will be the output if the user types 5 for the following code?

x = input("Enter a number: ")
print(x * 2)
Correct! Since x is a string "5", multiplying it by 2 repeats the string ("55").
Incorrect. Remember, input() returns a string. In Python, "5" * 2 results in "55", not 10.
