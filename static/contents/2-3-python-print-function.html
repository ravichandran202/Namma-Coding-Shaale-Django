Python print() Function

The print() function in Python is used to display output on the screen (usually the console).

It takes one or more values, converts them into strings (if they aren’t already), and prints them.
Example
# example of print function
print("Hello, World!")
 Output
                      Hello, World!
Syntax
# example of print function
print(*objects, sep=' ', end='\n')

  • objects : The value(s) to be printed (e.g., strings, numbers, variables).

  • sep=' ' : Separator between multiple objects (default: space ' ').

  • end='\n' : What to print at the end (default: newline \n).

1. Printing Multiple Items (Separated by Space)

# example of print function with multiple items separated by space
print("Hello", "World", "!")  
 Output
                      Hello, World!

2. Changing the Separator (sep)

Note : Items are separated by a space ( sep=' ' by default).

# example of print function with separator
print("Hello", "World", sep="-")
 Output
                                      Hello-World

3. Avoiding Newline (end='')

Note : The first print() ends with a new line by default.

# example of print function
print("Hello", end=' ')
print("World!")
 Output
                      Hello, World!
