String Methods

In Python, string methods are built-in functions that help you perform operations on strings such as changing case, searching, splitting, joining, or formatting text.

You call them using dot notation: "text".method()

Case Conversion
.upper()

Converts text to UPPERCASE

text = "hello"
print(text.upper()) 
 Output
HELLO
.lower()

Converts text to lowercase

text = "HELLO"
print(text.lower()) 
 Output
hello
.title()

Capitalizes first letter of each word

text = "hello world"
print(text.title()) 
 Output
Hello World
.capitalize()

Capitalizes first letter of sentence

text = "python programming"
print(text.capitalize()) 
 Output
Python programming
Whitespace Handling
.strip()

Removes spaces from both ends

text = "  hello  "
print(f"'{text.strip()}'") 
 Output
'hello'
.lstrip()

Removes spaces from left side only

text = "  hello  "
print(f"'{text.lstrip()}'") 
 Output
'hello  '
.rstrip()

Removes spaces from right side only

text = "  hello  "
print(f"'{text.rstrip()}'") 
 Output
'  hello'
Searching
.find()

Finds first position of text

text = "hello"
print(text.find("l")) 
 Output
2
.count()

Counts how many times text appears

text = "hello"
print(text.count("l")) 
 Output
2
.startswith()

Checks if string starts with text

text = "hello"
print(text.startswith("he")) 
 Output
True
.endswith()

Checks if string ends with text

text = "hello"
print(text.endswith("lo")) 
 Output
True
Replacing & Splitting
.replace()

Replaces text with other text

text = "hello"
print(text.replace("l", "x")) 
 Output
hexxo
.split()

Breaks text into list of words

text = "hello world"
print(text.split()) 
 Output
['hello', 'world']
.join()

Joins list into string with separator

words = ["a", "b", "c"]
print(",".join(words)) 
 Output
a,b,c
Validation
.isalpha()

True if only letters (a-z, A-Z)

print("abc".isalpha())
print("abc123".isalpha()) 
 Output
True
False
.isdigit()

True if only numbers (0-9)

print("123".isdigit())
print("12a".isdigit()) 
 Output
True
False
.isalnum()

True if only letters and numbers

print("abc123".isalnum())
print("abc 123".isalnum()) 
 Output
True
False
.isspace()

True if only spaces

print("   ".isspace())
print(" a ".isspace()) 
 Output
True
False
Formatting
.format()

Old way to put variables in strings

name = "Sam"
print("My name is {}".format(name)) 
 Output
My name is Sam
f"{}"

New way to put variables in strings

name = "Sam"
print(f"My name is {name}") 
 Output
My name is Sam
Length
len()

Counts characters in string

text = "hello"
print(len(text)) 
 Output
5

Quick Tips
  • All string methods return new strings (strings are immutable)
  • Use . (dot) notation: "text".method()
  • Methods with is prefix return True or False
  • len() is a function, not a method: len(text)
Try It Yourself

Use this string: text = " Python is FUN! "

Write code to:

  1. Remove spaces from both ends
  2. Convert to uppercase
  3. Replace "FUN" with "AWESOME"
  4. Split into words
  5. Count how many words
  6. Check if it starts with "Python"
text = "  Python is FUN!  "

# 1. Remove spaces
clean = text.strip()
print("1. No spaces:", clean)

# 2. Uppercase
upper_text = clean.upper()
print("2. Uppercase:", upper_text)

# 3. Replace
replaced = clean.replace("FUN", "AWESOME")
print("3. Replace:", replaced)

# 4. Split into words
words = clean.split()
print("4. Words:", words)

# 5. Count words
print("5. Word count:", len(words))

# 6. Check start
print("6. Starts with 'Python':", clean.startswith("Python")) 
 Output
1. No spaces: Python is FUN!
2. Uppercase: PYTHON IS FUN!
3. Replace: Python is AWESOME!
4. Words: ['Python', 'is', 'FUN!']
5. Word count: 3
6. Starts with 'Python': True
Remember
  • Use .method() after the string
  • Strings are immutable - methods return new strings
