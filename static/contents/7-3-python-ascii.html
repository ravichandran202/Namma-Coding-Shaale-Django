ASCII Character Encoding

ASCII is a numeric value given to different characters and symbols, for computers to store and manipulate.

ASCII stands for American Standard Code for Information Interchange.

Why ASCII?

Computers only understand numbers (0s and 1s). ASCII helps them understand text!

ASCII Value Examples
# ASCII Value Examples
print("ASCII value of 'A':", ord('A'))
print("ASCII value of 'B':", ord('B'))
print("ASCII value of 'a':", ord('a'))
print("ASCII value of '0':", ord('0')) 
 Output
ASCII value of 'A': 65
ASCII value of 'B': 66
ASCII value of 'a': 97
ASCII value of '0': 48
Key Observations
  • Uppercase letters start from 65 ('A')
  • Lowercase letters start from 97 ('a')
  • Numbers start from 48 ('0')
  • Difference between 'A' and 'a' is 32
  1. ord() - Find the Number for a Character

    Converts a character to its ASCII numeric value.

    # ord() function examples
print(ord('A'))  # Shows: 65
print(ord('!'))  # Shows: 33
print(ord('5'))  # Shows: 53 
 Output
65
33
53
    How ord() Works
    1. Takes a single character as input
    2. Returns the ASCII numeric value of that character
    3. Works with letters, numbers, and special characters
    4. Raises TypeError if input is not a single character
  2. chr() - Find the Character for a Number

    Converts an ASCII numeric value back to its character.

    # chr() function examples
print(chr(65))   # Shows: 'A'
print(chr(33))   # Shows: '!'
print(chr(53))   # Shows: '5' 
 Output
A
!
5
    How chr() Works
    1. Takes an integer (ASCII value) as input
    2. Returns the character corresponding to that ASCII value
    3. Valid range is typically 0-127 (standard ASCII)
    4. Raises ValueError if input is out of valid range

Practical Applications
# Application 1: Case conversion using ASCII
char = 'g'
# Convert lowercase to uppercase
if 'a' <= char <= 'z':
upper_char = chr(ord(char) - 32)
print(f"Uppercase of '{char}': '{upper_char}'")
 Output
Uppercase of 'g': 'G'
# Application 2: Character type detection
character = '7'
ascii_val = ord(character)
if 48 <= ascii_val <= 57:
    print(f"'{character}' is a digit")
elif 65 <= ascii_val <= 90:
    print(f"'{character}' is uppercase letter")
elif 97 <= ascii_val <= 122:
    print(f"'{character}' is lowercase letter")
else:
    print(f"'{character}' is a special character") 
 Output
'7' is a digit
How It Works
  • Case Conversion: Subtract 32 from lowercase ASCII to get uppercase
  • Character Detection: Check ASCII ranges to determine character type
  • Pattern: ord() → manipulate → chr()
Practice Exercise

Write a Python program that:

  1. Takes a character as input
  2. Prints its ASCII value
  3. Converts it to uppercase if it's lowercase, and vice versa
  4. Prints the converted character and its ASCII value

Use ASCII values and the ord()/chr() functions.

# Solution: Character case converter using ASCII
char = 'm'  # Example input

# Get ASCII value
ascii_val = ord(char)
print(f"Original: '{char}' → ASCII: {ascii_val}")

# Case conversion using ASCII
if 65 <= ascii_val <= 90:  # Uppercase
    converted_char = chr(ascii_val + 32)
    print(f"Converted: '{converted_char}' → ASCII: {ord(converted_char)}")
elif 97 <= ascii_val <= 122:  # Lowercase
    converted_char = chr(ascii_val - 32)
    print(f"Converted: '{converted_char}' → ASCII: {ord(converted_char)}")
else:
    print("Not a letter - no conversion needed") 
 Output
Original: 'm' → ASCII: 109
Converted: 'M' → ASCII: 77
Step-by-Step Solution
  1. Get ASCII value of input character using ord()
  2. Check if it's uppercase (65-90) or lowercase (97-122)
  3. Convert case by adding/subtracting 32 from ASCII value
  4. Use chr() to get the converted character
  5. Print both original and converted characters with their ASCII values
