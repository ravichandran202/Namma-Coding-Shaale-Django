Python Random Module

The random module is built into Python and helps you generate pseudo-random numbers. It's essential for games, simulations, and testing.

Important Step
Before using these functions, you must import the module by adding import random at the top of your file.

Common Functions

random.randint(a, b)

Returns a random integer $N$ such that $a \le N \le b$ (both endpoints are included).

import random

# Roll a die (1 to 6)
dice_roll = random.randint(1, 6)
print(f"You rolled a: {dice_roll}")
random.random()

Returns a random float number between 0.0 and 1.0.

import random

prob = random.random()
print(prob)  # e.g., 0.5392...
random.uniform(a, b)

Returns a random float between a and b.

price = random.uniform(10.5, 20.5)
print(f"Random Price: ${price:.2f}")
random.choice(seq)

Picks one random element from a list or string.

import random

fruits = ["Apple", "Banana", "Cherry"]
print(random.choice(fruits))
random.shuffle(seq)

Shuffles the list in place (modifies the original list).

cards = ["A", "K", "Q", "J"]
random.shuffle(cards)
print(cards) # e.g., ['Q', 'A', 'J', 'K']

Controlling Randomness

Sometimes you need the "random" numbers to be the same every time you run the code (e.g., for debugging). You can use random.seed().

import random

random.seed(10)
print(random.randint(1, 100))  # Will always print 74 (or specific number for this seed)

Mini Challenge: The Lucky Draw

Write a program that:

  1. Defines a list of 3 prizes: "Car", "Bike", "Cash".
  2. Randomly picks one prize.
  3. Prints "Congratulations! You won a [PRIZE]".
import random

prizes = ["Car", "Bike", "Cash"]
winner = random.choice(prizes)

print(f"Congratulations! You won a {winner}")
