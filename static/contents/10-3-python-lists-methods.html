Complete List Methods

All built-in methods for working with lists in Python.

.append()

Adds one element to the end of the list

nums = [1, 2]
nums.append(3)
print(nums) 
 Output
[1, 2, 3]
.clear()

Removes all elements from the list

nums = [1, 2, 3]
nums.clear()
print(nums) 
 Output
[]
.copy()

Creates a copy of the list

nums = [1, 2, 3]
new_nums = nums.copy()
print(new_nums) 
 Output
[1, 2, 3]
.count()

Counts how many times a value appears

nums = [1, 2, 2, 3]
print(nums.count(2)) 
 Output
2
.extend()

Adds multiple elements to the end

nums = [1, 2]
nums.extend([3, 4])
print(nums) 
 Output
[1, 2, 3, 4]
.index()

Finds position of first occurrence

nums = [10, 20, 30]
print(nums.index(20)) 
 Output
1
.insert()

Inserts element at specific position

nums = [1, 3]
nums.insert(1, 2)  # Insert at index 1
print(nums) 
 Output
[1, 2, 3]
.pop()

Removes and returns element at position

nums = [1, 2, 3]
removed = nums.pop()  # Removes last element
print("Removed:", removed)
print("List:", nums) 
 Output
Removed: 3
List: [1, 2]
# Pop from specific position
nums = [1, 2, 3, 4]
removed = nums.pop(1)  # Remove element at index 1
print("Removed:", removed)
print("List:", nums) 
 Output
Removed: 2
List: [1, 3, 4]
.remove()

Removes first occurrence of value

nums = [1, 2, 3, 2]
nums.remove(2)  # Removes first '2'
print(nums) 
 Output
[1, 3, 2]
.reverse()

Reverses the order of elements

nums = [1, 2, 3]
nums.reverse()
print(nums) 
 Output
[3, 2, 1]
.sort()

Sorts elements in order

nums = [3, 1, 2]
nums.sort()
print(nums) 
 Output
[1, 2, 3]
# Sort in reverse order
nums = [3, 1, 2]
nums.sort(reverse=True)
print(nums) 
 Output
[3, 2, 1]

Try It Yourself

Start with: numbers = [5, 2, 8, 2, 1]

Use list methods to:

  1. Add 9 to the end
  2. Insert 4 at position 2
  3. Remove the first 2
  4. Count how many 2's
  5. Sort the list
  6. Reverse the list
  7. Remove the last element
  8. Clear the list
numbers = [5, 2, 8, 2, 1]

# 1. Add 9 to end
numbers.append(9)
print("1. After append:", numbers)

# 2. Insert 4 at position 2
numbers.insert(2, 4)
print("2. After insert:", numbers)

# 3. Remove first 2
numbers.remove(2)
print("3. After remove:", numbers)

# 4. Count 2's
count = numbers.count(2)
print("4. Count of 2's:", count)

# 5. Sort the list
numbers.sort()
print("5. After sort:", numbers)

# 6. Reverse the list
numbers.reverse()
print("6. After reverse:", numbers)

# 7. Remove last element
numbers.pop()
print("7. After pop:", numbers)

# 8. Clear the list
numbers.clear()
print("8. After clear:", numbers) 
 Output
1. After append: [5, 2, 8, 2, 1, 9]
2. After insert: [5, 2, 4, 8, 2, 1, 9]
3. After remove: [5, 4, 8, 2, 1, 9]
4. Count of 2's: 1
5. After sort: [1, 2, 4, 5, 8, 9]
6. After reverse: [9, 8, 5, 4, 2, 1]
7. After pop: [9, 8, 5, 4, 2]
8. After clear: []
Key Points to Remember
  • Adding: Use .append() for one item, .extend() for multiple
  • Removing: Use .remove() by value, .pop() by position
  • Finding: Use .index() for position, .count() for occurrences
  • Ordering: Use .sort() to sort, .reverse() to reverse
  • Copying: Use .copy() to avoid modifying original list
  • Clearing: Use .clear() to empty the list
  • Most list methods modify the original list directly
