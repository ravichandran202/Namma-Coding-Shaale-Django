2D Lists (Nested Lists)

2D lists are lists that contain other lists - like tables with rows and columns.

What is a 2D List?

A 2D list is a list where each element is another list.

Think of it as a table with rows and columns.

Syntax
# A 2D list of numbers (like a 3x3 table)
    list_name = [
    [row1_item1, row1_item2, ...],
    [row2_item1, row2_item2, ...],
    ...
]
Example
matrix = [
    [1, 2, 3],     # Row 0
    [4, 5, 6],     # Row 1
    [7, 8, 9]      # Row 2
]
print(matrix)
 Output
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]

1. Creating 2D Lists

# A 2D list of numbers (like a 3x3 table)
matrix = [
    [1, 2, 3],     # Row 0
    [4, 5, 6],     # Row 1
    [7, 8, 9]      # Row 2
]
print("2D List:", matrix) 
 Output
2D List: [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
# Mixed data types
people = [
    ["Alice", 25, True],
    ["Bob", 30, False],
    ["Charlie", 35, True]
]
print("People data:", people) 
 Output
People data: [['Alice', 25, True], ['Bob', 30, False], ['Charlie', 35, True]]

2. Accessing Elements

Use two indices: [row][column]

matrix = [
    [1, 2, 3],   # Row 0
    [4, 5, 6],   # Row 1
    [7, 8, 9]    # Row 2
]

# Access single elements
print("Row 0, Column 1:", matrix[0][1])  # 2
print("Row 2, Column 0:", matrix[2][0])  # 7
print("Row 1, Column 2:", matrix[1][2])  # 6

# Access entire rows
print("Row 1:", matrix[1])     # [4, 5, 6]
print("Row 2:", matrix[2])     # [7, 8, 9] 
 Output
Row 0, Column 1: 2
Row 2, Column 0: 7
Row 1, Column 2: 6
Row 1: [4, 5, 6]
Row 2: [7, 8, 9]

3. Looping Through 2D Lists

Method 1: Loop Through Rows

Get each row as a list

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("Each row:")
for row in matrix:
    print(row) 
 Output
Each row:
[1, 2, 3]
[4, 5, 6]
[7, 8, 9]
Method 2: Nested Loop for All Elements

Get each individual element

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("All elements:")
for row in matrix:
    for item in row:
        print(item, end=" ")
    print()  # New line after each row 
 Output
All elements:
1 2 3 
4 5 6 
7 8 9
Method 3: Using Row and Column Indices

When you need both position and value

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("With positions:")
for row_index in range(len(matrix)):
    for col_index in range(len(matrix[row_index])):
        print(f"matrix[{row_index}][{col_index}] = {matrix[row_index][col_index]}") 
 Output
With positions:
matrix[0][0] = 1
matrix[0][1] = 2
matrix[0][2] = 3
matrix[1][0] = 4
matrix[1][1] = 5
matrix[1][2] = 6
matrix[2][0] = 7
matrix[2][1] = 8
matrix[2][2] = 9

Practical Examples

# Student grades example
grades = [
    ["Alice", 85, 92, 78],
    ["Bob", 76, 88, 95],
    ["Charlie", 92, 90, 96]
]

print("Student Grades:")
for student in grades:
    name = student[0]
    scores = student[1:]
    average = sum(scores) / len(scores)
    print(f"{name}: {scores} → Average: {average:.1f}") 
 Output
Student Grades:
Alice: [85, 92, 78] → Average: 85.0
Bob: [76, 88, 95] → Average: 86.3
Charlie: [92, 90, 96] → Average: 92.7
# Tic-tac-toe board
board = [
    ["X", "O", " "],
    [" ", "X", "O"],
    ["O", " ", "X"]
]

print("Tic-Tac-Toe Board:")
for row in board:
    print(" | ".join(row))
    print("-" * 9) 
 Output
Tic-Tac-Toe Board:
X | O |  
---------
  | X | O
---------
O |   | X
---------

Try It Yourself

Create a 2D list for a 2×3 grid:

1  2  3
4  5  6

Then:

  1. Print the entire 2D list
  2. Access and print the element at row 1, column 2
  3. Loop through and print each row
  4. Loop through and print all elements with positions
  5. Calculate the sum of all numbers
# Create the 2D list
grid = [
    [1, 2, 3],
    [4, 5, 6]
]

# 1. Print entire list
print("1. Entire list:", grid)

# 2. Access specific element
print("2. Element at [1][2]:", grid[1][2])

# 3. Print each row
print("3. Each row:")
for row in grid:
    print(row)

# 4. Print all elements with positions
print("4. All elements with positions:")
for i in range(len(grid)):
    for j in range(len(grid[i])):
        print(f"grid[{i}][{j}] = {grid[i][j]}")

# 5. Calculate sum
total = 0
for row in grid:
    for num in row:
        total += num
print("5. Sum of all numbers:", total) 
 Output
1. Entire list: [[1, 2, 3], [4, 5, 6]]

2. Element at [1][2]: 6

3. Each row:
[1, 2, 3]
[4, 5, 6]

4. All elements with positions:
grid[0][0] = 1
grid[0][1] = 2
grid[0][2] = 3
grid[1][0] = 4
grid[1][1] = 5
grid[1][2] = 6

5. Sum of all numbers: 21
Remember
  • 2D lists are lists containing other lists
  • Use [row][column] to access elements
  • First index = row, Second index = column
  • Use nested loops to go through all elements
  • Great for representing tables, grids, and matrices
  • Each inner list can have different lengths
