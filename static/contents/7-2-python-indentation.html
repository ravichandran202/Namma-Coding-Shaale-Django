Python Indentation

Space(s) in front of the conditional block is called indentation.

Indentation is required to define code blocks instead of using curly braces {} like other languages.

Key Points About Indentation
  • Standard practice is to use four spaces for indentation
  • Codes with the same indentation level belong to the same block
  • Indentation determines which statements are part of which code block
  • Inconsistent indentation causes IndentationError
Correct Indentation Example
# Example: Proper indentation with conditional statements
age = 18
if age >= 18:
    print("You are an adult")  # 4 spaces
    print("You can vote")      # 4 spaces
print("This always runs")      # 0 spaces 
 Output
 You are an adult
You can vote
This always runs
Step-by-Step Evaluation
  1. Variable age is assigned value 18
  2. Condition age >= 18 is evaluated: 18 >= 18 → True
  3. Both indented statements under if execute (same indentation level)
  4. The last statement (no indentation) always executes
Indentation Error Example
# Example: Incorrect indentation causes error
age = 18
if age >= 18:
    print("You are an adult")  # 4 spaces
   print("You can vote")       # 3 spaces - ERROR!
print("This always runs")      # 0 spaces 
 Output
 IndentationError: unexpected indent
Error Analysis
  1. First print statement has 4 spaces of indentation
  2. Second print statement has only 3 spaces of indentation
  3. Python detects inconsistent indentation
  4. Raises IndentationError: unexpected indent
  5. Program stops execution immediately
More Indentation Examples
# Example: Nested conditional statements
number = 15
if number > 0:
    print("Positive number")        # 4 spaces
    if number % 2 == 0:             # 4 spaces
        print("Even number")        # 8 spaces
    else:                           # 4 spaces
        print("Odd number")         # 8 spaces
else:
    print("Non-positive number")    # 4 spaces
print("Program complete")           # 0 spaces 
 Output
 Positive number
Odd number
Program complete
Nested Indentation Levels
  • Level 0: Main program flow
  • Level 1 (4 spaces): Outer if-else blocks
  • Level 2 (8 spaces): Inner if-else blocks
# Example: Multiple elif statements
score = 85
if score >= 90:
    print("Grade: A")           # 4 spaces
    print("Excellent!")         # 4 spaces
elif score >= 80:
    print("Grade: B")           # 4 spaces
    print("Good job!")          # 4 spaces
elif score >= 70:
    print("Grade: C")           # 4 spaces
    print("Well done!")         # 4 spaces
else:
    print("Grade: F")           # 4 spaces
    print("Need improvement")   # 4 spaces
print("Grading complete")       # 0 spaces 
 Output
Grade: B
Good job!
Grading complete
Python Indentation Rules
Rule Description Example
Consistency All statements in same block must have same indentation 4 spaces throughout
Increase for nesting Add indentation for each nested level 4 → 8 → 12 spaces
Decrease for unnesting Reduce indentation when exiting blocks 12 → 8 → 4 spaces
No mixing Don't mix spaces and tabs Use only spaces
Standard PEP 8 recommends 4 spaces per level 4 spaces per indent
Common Indentation Errors
Error Type Cause Solution
IndentationError Inconsistent spacing in same block Use consistent 4-space indentation
TabError Mixing tabs and spaces Use spaces only, configure editor
SyntaxError Missing indentation after colon Add proper indentation after :
Unexpected indent Indentation without preceding block Remove unnecessary indentation
