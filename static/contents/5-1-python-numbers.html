Python Numbers

Python has several built-in numeric data types for representing different kinds of numbers.

Python Number Types

1. Integer (int):

  • Whole numbers (positive or negative) without decimal points

  • Unlimited precision (can be very large)

    • # example of python numeric datatypes
x = 10
y = -5
z = 12345678901234567890

print(x)
print(y)
print(z)

print(f"type of {x} is {type(x)}")
print(f"type of {y} is {type(y)}")
print(f"type of {z} is {type(z)}")

 Output
                   10
-5
12345678901234567890
type of 10 is <class 'int'>
type of -5 is <class 'int'>
type of 12345678901234567890 is <class 'int'>

2. Floating Point (float)

  • Numbers with decimal points.

  • Represent real numbers but have limited precision

  • Can also use scientific notation

    • # example of python floating datatype
a = 3.14
b = -0.001
c = 2.5e3  # 2500.0
d = 1.6e-4 # 0.00016

print(a)
print(b)
print(c)
print(d)

print(f"type of {a} is {type(a)}")
print(f"type of {b} is {type(b)}")
print(f"type of {c} is {type(c)}")
print(f"type of {d} is {type(d)}")

 Output
                   3.14
-0.001
2500.0
0.00016
type of 3.14 is <class 'float'>
type of -0.001 is <class 'float'>
type of 2500.0 is <class 'float'>
type of 0.00016 is <class 'float'>

    Explanation: e means "exponent" and is followed by the power of 10.

3. Complex Numbers

  • Numbers with real and imaginary parts.

  • The imaginary part is denoted with jor J

    • # example of python complex number datatype
c1 = 3 + 4j
c2 = -2j
c3 = complex(1, 2)  # 1+2j

print(c1)
print(c2)
print(c3)

print(f"type of {c1} is {type(c1)}")
print(f"type of {c2} is {type(c2)}")
print(f"type of {c3} is {type(c3)}")

 Output
                   (3+4j)
(-0-2j)
(1+2j)
type of (3+4j) is <class 'complex'>
type of (-0-2j) is <class 'complex'>
type of (1+2j) is <class 'complex'>

    Explanation:

    Two dimensions in one number: Compact representation of 2D systems

    Complex numbers model anything with two related components - position (x,y),

Python Numeric Datatype Conversion

Type conversion (also called type casting) allows you to convert between different types in Python.

1. Integer (int):

  • use int() function to convert values to integers.

  • Converting float to integer truncates (removes) the decimal part

  • Boolean values have specific numeric equivalents

  • Strings can be converted only if they contain valid numeric values

  • For decimal strings, convert to float first, then to int

    • # int() - to integer
var1 = 3.14  #float data type
var2 = True  #boolean data type
var3 = "100" #string data type

print(int(var1))     # 3
print(int(var2))     # 1
print(int(var3))     # 100

print(f"Before conversion : {var1}, After Conversion : {int(var1)}")
print(f"Before conversion : {var2}, After Conversion : {int(var2)}")
print(f"Before conversion : {var3}, After Conversion : {int(var3)}")

 Output
                   3
1
100
Before conversion : 3.14, After Conversion : 3
Before conversion : True, After Conversion : 1
Before conversion : 100, After Conversion : 100

2. Float (float):

  • use float() function to convert values to floating numbers.

  • Converting integer to float adds decimal point

  • Boolean values convert to their float equivalents

  • Strings can be converted if they contain valid numeric values

  • Python supports special float values : float("inf"), float("-inf"), float("nan")

    • # int() - to integer
var1 = 5      # integer data type
var2 = False  # boolean data type
var3 = "3.14" # string data type

print(float(var1))     # 5.0
print(float(var2))     # 0.0
print(float(var3))     # 3.14

print(f"Before conversion : {var1}, After Conversion : {float(var1)}")
print(f"Before conversion : {var2}, After Conversion : {float(var2)}")
print(f"Before conversion : {var3}, After Conversion : {float(var3)}")

 Output
                   5.0
0.0
3.14
Before conversion : 5, After Conversion : 5.0
Before conversion : False, After Conversion : 0.0
Before conversion : 3.14, After Conversion : 3.14

Special Floating Numbers

These represent special mathematical concepts that regular numbers can't express:

  1. Positive Infinity : Represents a value that is greater than any finite number.

    Syntax : float("inf")

  2. Negative Infinity : Represents a value that is less than any finite number.

    Syntax : float("-inf")

  3. Not a Number : Represents a value of undefined or unrepresentable numerical results.

    Syntax : float("nan")

    4. # examples for special floating numbers
var1 = float("inf")   # positive infinity
var2 = float("-inf")  # negative infinity
var3 = float("nan")   # not a number

print(var1)
print(var2)
print(var3)

 Output
                   inf
-inf
nan

Summary

  • Python provides three main numeric data types — int, float, and complex.
  • int represents whole numbers without decimals and supports unlimited precision.
  • float represents real numbers with decimal points or scientific notation (e.g., 2.5e3).
  • complex numbers have real and imaginary parts (e.g., 3 + 4j).
  • Python allows type conversion using functions like int() and float().
  • Converting float → int removes decimals, and bool → int gives 1 for True and 0 for False.
  • Special float values include inf (infinity), -inf (negative infinity), and nan (not a number).
  • Python automatically handles large integers and floating-point precision internally.
