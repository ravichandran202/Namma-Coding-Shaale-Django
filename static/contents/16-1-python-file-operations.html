Python File Handling

File Handling in Python is the process of creating, opening, reading, writing, appending, and closing files using Python's built-in functions.

Python File Operations

Python provides built-in functions to work with files.

Basic Steps: Open a file → Read/Write → Close the file

1. Opening a File

# Use the open() function
file = open(file_name, operation_type)
File Modes in Python
Mode Description Example / Usage
"r" Read (default). File must exist. open("file.txt", "r")
"w" Write. Creates new file or overwrites existing. open("file.txt", "w")
"a" Append. Adds data at the end. open("file.txt", "a")
"x" Create new file. Error if file exists. open("file.txt", "x")
"b" Binary mode (for images, audio, etc.) open("file.jpg", "rb")
"t" Text mode (default). open("file.txt", "rt")

2. Reading a File

f = open("data.txt", "r")

print(f.read())        # read entire file
print(f.read(5))       # read first 5 characters
print(f.readline())    # read one line
print(f.readlines())   # read all lines as list

f.close()

3. Writing to a File

Write Mode - Creates new file or overwrites existing
f = open("data.txt", "w")
f.write("First line\n")
f.write("Second line\n")
f.close()
Append Mode - Adds data at the end
f = open("data.txt", "a")
f.write("Extra line\n")
f.close()

4. Closing a File

Important

Always close files to free resources:

f.close()

Safe File Handling with try-finally
f = None
try:
    f = open("test.txt", "r")
    content = f.read()
    print(content)
except Exception as e:
    print(f"Error: {e}")
else:
   if f is not None:
      content = f.read()
      print(content)
finally:
    if f is not None:
        f.close()

Using with Statement (Best Practice)

Recommended Approach

No need to call close() manually. File closes automatically.

with open("data.txt", "r") as f:
    data = f.read()
    print(data)

Reading and Writing Examples

# Write to file
with open("students.txt", "w") as f:
    f.write("Alice\n")
    f.write("Bob\n")

# Read file
with open("students.txt", "r") as f:
    print(f.readlines()) 
 Output
['Alice\n', 'Bob\n']
Key Points
  • Use open() with appropriate mode ("r", "w", "a")
  • Always close files with close() or use with statement
  • read() - read entire file, readline() - read one line
  • readlines() - read all lines as list
  • write() - write data to file
  • Use with statement for automatic file closing
