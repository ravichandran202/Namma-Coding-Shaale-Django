String Indexing & Slicing

1. Introduction to String Indexing

In Python, a string is an ordered sequence of characters. Indexing allows you to access a specific character by its position.

The Golden Rule of Python Indexing

Counting always starts from 0 (zero), not 1. The first character is at index 0.

# Example string for our visualization
word = "PYTHON"
Visual Index Map: "PYTHON"
Character P Y T H O N
Positive Index 0 1 2 3 4 5
Negative Index -6 -5 -4 -3 -2 -1
Notice how Positive indices count forward (0 to 5) and Negative indices count backward (-1 to -6).

Method A: Positive Indexing (Left to Right)

Use positive integers [0, 1, 2...] to access characters starting from the beginning of the string.

word = "PYTHON"

first_letter = word[0]  # Accesses 'P'
third_letter = word[2]  # Accesses 'T'

print(first_letter)
print(third_letter) 
 Output
P
T

Method B: Negative Indexing (Right to Left)

Python has a special feature called Negative Indexing. This allows you to access characters starting from the end of the string.

Why use it?
Imagine you have a very long string and you just want the last character. You don't need to count the length! Just use -1.
  • index -1 refers to the Last item.
  • index -2 refers to the Second to last item.
  • and so on...
word = "PYTHON"

print(word[-1])  # 'N' (The very last character)
print(word[-2])  # 'O' (The second to last character)
print(word[-6])  # 'P' (Going all the way back to start) 
 Output
N
O
P

2. String Slicing (Extracting Substrings)

Slicing returns a range of characters. It works like a cookie cutter, extracting a piece of the original string.

Syntax: string[ start : stop : step ]
1. Start
The index where the slice begins. This is Included.
2. Stop
The index where the slice ends. This is Excluded (upto, but not including).
3. Step (Optional)
The jump size. Default is 1 (take every character).

Basic Slicing Examples

text = "PROGRAMMING"

# Get "PRO" (Indices 0, 1, 2)
# We stop at 3, because 3 is excluded
print(text[0:3])

# Get "GRAM" (Indices 3, 4, 5, 6)
print(text[3:7]) 
 Output
PRO
GRAM

Negative Slicing (Slicing from the End)

You can use negative numbers to slice the string relative to the end. This is extremely useful for removing suffixes or grabbing the last few characters.

text = "PROGRAMMING"

# Example 1: Get the last 3 characters
# Start at -3 (M) and go to the end (:)
print(text[-3:])

# Example 2: Remove the last 3 characters
# Start at beginning (:) and stop at -3 (excluding last 3)
print(text[:-3])

# Example 3: Get "GRAM" using negative indices
# "GRAM" corresponds to indices -8 to -4
print(text[-8:-4]) 
 Output
ING
PROGRAMM
GRAM

Using the 'Step' Argument

The third number defines the "jump".

numbers = "0123456789"

print(numbers[::2])   # Take every 2nd character (0, 2, 4, 6, 8)
print(numbers[::3])   # Take every 3rd character (0, 3, 6, 9)

# THE COOLEST TRICK: REVERSING A STRING
# By using a negative step (-1), we walk backwards!
print(numbers[::-1])  # "9876543210"
Summary Cheat Sheet
s[0] First character
s[-1] Last character
s[-3:] Last 3 characters
s[:-1] Everything except the last character
s[::-1] Reverses the string
