While Loops in Python

While loops allow us to execute a block of code as long as a condition remains True.

While Loop Structure 
# Syntax
# Initialization
while condition:
    # code to execute for each iteration
    # updation
Basic While Loop Example
# Program to print numbers from 1 to 5
counter = 1          # Initialization
while counter <= 5:  # Condition
    print(counter)
    counter += 1     # Updation 
 Output
1
2
3
4
5
Step-by-Step Execution
  1. Initialization: counter = 1
  2. Check condition: 1 <= 5 → True
  3. Execute body: Print 1
  4. Update: counter = 2
  5. Repeat until condition becomes False
  6. Exit: When counter = 6, condition becomes False

Common While Loop Mistakes

  1. 1. Missing Initialization

    Forgetting to initialize the loop control variable before the while loop.

    # Incorrect: Missing initialization
num = 5
while counter <= 5:
    print(counter)
    counter += 1 
 Output
NameError: name 'counter' is not defined
    #  Correct: Proper initialization
counter = 1        # Initialize before loop
num = 5
while counter <= 5:
    print(counter)
    counter += 1 
 Output
1
2
3
4
5
    Error Analysis
    • Python doesn't know what counter is
    • Variable must be defined before use
    • NameError occurs when using undefined variables
  2. 2. Incorrect Indentation

    Forgetting to indent the code block inside the while loop.

    # Incorrect: Missing indentation
counter = 5
while counter <= 5:
print(counter)     # This will cause an IndentationError
    counter += 1 
 Output
IndentationError: expected an indented block
    #  Correct: Proper indentation
counter = 5
while counter <= 5:
    print(counter)     # 4 spaces indentation
    counter += 1       # 4 spaces indentation 
 Output
5
    Error Analysis
    • Python uses indentation to define code blocks
    • All statements inside loop must be indented
    • Standard practice: 4 spaces per indentation level
  3. 3. Using = instead of == in Conditions

    Confusing assignment operator (=) with equality operator (==).

    # Incorrect: Using assignment in condition
counter = 0
while counter = 5:  # SyntaxError, should be ==
    print(counter)
    counter += 1 
 Output
SyntaxError: invalid syntax
    #  Correct: Using equality comparison
counter = 0
while counter == 5:  # Correct equality operator
    print(counter)
    counter += 1 
 Output
# No output (condition is False initially)
    Error Analysis
    • = is for assignment (storing values)
    • == is for comparison (checking equality)
    • Using = in conditions causes SyntaxError
  4. 4. Forgetting the Colon

    Missing the colon (:) at the end of the while statement.

    # Incorrect: Missing colon
counter = 0
while counter == 5  # SyntaxError, should add :
    print(counter)
    counter += 1 
 Output
SyntaxError: invalid syntax
    #  Correct: With colon
counter = 0
while counter == 5:  # Colon present
    print(counter)
    counter += 1 
 Output
# No output (condition is False initially)
    Error Analysis
    • Colon (:) indicates the start of a code block
    • Required after while, for, if, def, etc.
    • Missing colon causes SyntaxError
  5. 5. Infinite Loops

    Forgetting to update the loop control variable, causing the loop to run forever.

    # Incorrect: Infinite loop
count = 1
while count <= 5:
    print(count)
    # Forgot to increment count! 
 Output
1
1
1
1
... (runs forever until manually stopped)
    # Correct: Proper updation
count = 1
while count <= 5:
    print(count)
    count += 1  # Increment to avoid infinite loop 
 Output
1
2
3
4
5
    Error Analysis
    • Condition count <= 5 always remains True
    • No updation means no progress toward exit condition
    • Program runs indefinitely until interrupted

While Loop Best Practices
Practice Description Example
Initialize before loop Define loop variable with starting value counter = 1
Use clear condition Condition should eventually become False while counter <= 10
Always update variable Modify loop variable inside the loop counter += 1
Test with small values Debug with small iterations first while counter <= 3
