Python Variables

A variable is a named storage location in memory that holds a value. It allows data to be stored, modified, and retrieved in a program.

Key Properties:

  • Must have a unique identifier (name).

  • Has a data type (e.g., int,float,str).

  • Can be reassigned (in most languages).

Example in python
# example of python variables
age = 30           # 'age' is the variable storing value 30
name = "Bob"       # 'name' stores a string
price = 45.99      # 'price' stores a float
Example in C
// example of c variables
int age = 5;        // 'age' is an integer variable
float price = 3.14; // 'price' is a float variable
char letter = 'A';  // 'letter' is a character variable
Example Multiple Assignment

Python allows you to assign values to multiple variables in a single line. This is not just a shortcut; it’s an efficient way to handle data.

# Assigning different values to multiple variables
a, b = 10, 20

# Assigning the same value to multiple variables
x = y = z = 100

  • How it works: Python evaluates the expressions on the right side first, puts them in a temporary structure (a tuple), and then assigns them to the variables on the left.

  • Memory Impact: In a, b = 10, 20, two distinct objects are created in memory, and the symbol table is updated so a points to 10 and b points to 20.

Pro Tip: Swapping Values
In many languages, you need a temp variable to swap two values. In Python, you can simply do: a, b = b, a.

What Happens in Memory

A. Variable Creation
# example of python variable creation
age = 25

When you declare a variable like age = 25

  • Memory Allocation:

    • The system finds an available memory slot (e.g., at address 0x1000).

    • It stores 25 in that location..

  • Symbol Table Update:

    • Python maintains a symbol table (a dictionary-like structure) that maps age→ 0x1000.

B. Variable Assignment (Changing Value)
# example of python variable creation
age = 25
age = 30  # here we are changing the value for the same variable

  • The system checks the symbol table for age.

  • It updates the value at 0x1000 from 25 to 30.

C. Multiple Variables Pointing to the Same Value
# example of multiple variable pointing to the same value
age = 25
b = a  # 'b' now points to the same value as 'a'

  • Both a and b refer to the same memory address (0x1000 where 25 is stored).

  • This is called aliasing (two names for the same object).

Three Ways to Swap Variables

Swapping means interchanging the values of two variables. For example, if a = 5 and b = 10, after swapping, a should be 10 and b should be 5.

1. Using a Temporary Variable (Classic Way)

This is the most common method used in languages like C or Java. It uses a "third person" (temp) to hold a value during the swap.

a = 5
b = 10

temp = a  # temp now holds 5
a = b     # a now holds 10
b = temp  # b now holds 5 (from temp)
2. Using Arithmetic (No Extra Space)

You can swap numbers without a third variable by using addition and subtraction. This is a clever logic trick.

a = 5
b = 10

a = a + b  # a becomes 15
b = a - b  # b becomes 5 (15 - 10)
a = a - b  # a becomes 10 (15 - 5)
3. The Pythonic Way (Multiple Assignment)

Python makes this incredibly simple in a single line using tuple unpacking. This is the recommended way in Python.

a = 5
b = 10

# The Magic Line
a, b = b, a

print(a) # Output: 10
print(b) # Output: 5
Why Method 3 is best?
The Pythonic method (a, b = b, a) is not just shorter; it is also safer. It evaluates the right-hand side (the values) before assigning them to the left-hand side, preventing the data-loss issues you might face if you write manual logic incorrectly.

Summary

  • A variable is a named memory location that stores a value which can be accessed and modified in a program.
  • Each variable has a name (identifier), a data type (int, float, str), and a memory address.
  • When a variable is created, memory is allocated and mapped in the symbol table.
  • Reassigning a variable updates its stored value in memory.
  • Two variables can refer to the same value in memory — this is called aliasing.
