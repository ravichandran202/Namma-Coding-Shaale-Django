Python Variables

A variable is a named storage location in memory that holds a value. It allows data to be stored, modified, and retrieved in a program.

Key Properties:

  • Must have a unique identifier (name).

  • Has a data type (e.g., int,float,str).

  • Can be reassigned (in most languages).

Example in python
# example of python variables
age = 30           # 'age' is the variable storing value 30
name = "Bob"       # 'name' stores a string
price = 45.99      # 'price' stores a float
Example in C
// example of c variables
int age = 5;        // 'age' is an integer variable
float price = 3.14; // 'price' is a float variable
char letter = 'A';  // 'letter' is a character variable

How Variables Are Stored in Memory

reference - datamentor.io

  • age is the variable name (identifier).

  • 25 is the value stored in memory.

  • The computer assigns a memory address (like 0x7ffd42a3b5f0) to store 25.

What Happens in Memory

A. Variable Creation
# example of python variable creation
age = 25

When you declare a variable like age = 25

  • Memory Allocation:

    • The system finds an available memory slot (e.g., at address 0x1000).

    • It stores 25 in that location..

  • Symbol Table Update:

    • Python maintains a symbol table (a dictionary-like structure) that maps age→ 0x1000.

B. Variable Assignment (Changing Value)
# example of python variable creation
age = 25
age = 30  # here we are changing the value for the same variable

  • The system checks the symbol table for age.

  • It updates the value at 0x1000 from 25 to 30.

C. Multiple Variables Pointing to the Same Value
# example of multiple variable pointing to the same value
age = 25
b = a  # 'b' now points to the same value as 'a'

  • Both a and b refer to the same memory address (0x1000 where 25 is stored).

  • This is called aliasing (two names for the same object).

Summary

  • A variable is a named memory location that stores a value which can be accessed and modified in a program.
  • Each variable has a name (identifier), a data type (int, float, str), and a memory address.
  • When a variable is created, memory is allocated and mapped in the symbol table.
  • Reassigning a variable updates its stored value in memory.
  • Two variables can refer to the same value in memory — this is called aliasing.
