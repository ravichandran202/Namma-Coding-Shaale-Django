Introduction To Programming

Basic Definitions

1. Software

Software is instructions that tell your computer what to do.

Types :

  1. System Software : Manages your computer (Windows, Linux)

  2. Application Software : Programs you use daily (Chrome, Microsoft Word)

2. Hardware

Physical parts of a computer, like the CPU, memory, keyboard, and screen..

3. Programming

Programming means writing instructions to create software.

4. Coding

The instructions that we write to create software are called Code..

5. Syntax

Grammar rules for each programming language (like English has grammar rules).

6. Algorithms

Step-by-step instructions to solve problems (like a cooking recipe).

7. Data, Input & Output

  1. Data : Data is a collection of facts, numbers, words or other useful information.

  2. Input : Data going into program (user clicks, file data).

  3. Output : Results from program (website display, saved file).

Stages of Software Development

Building a software app involves different steps, from the first idea to maintaining it after it's released.

  1. Design : Plan what your program will do

  2. Coding : Write the program using a programming language

  3. Testing : Check if your program works correctly

  4. Deployment & Maintenance : Put your program online and keep it updated

Popular Programming Languages

Building a software app involves different steps, from the first idea to maintaining it after it's released.

  • Web : HTML, CSS, JavaScript, Python

  • Mobile Apps : Java, Swift

  • Games : C#, C++

  • Data Analysis : Python, R

Note : It's important to know that HTML and CSS are not programming languages. They are markup languages used to structure and style web pages, but they don't have the logic and programming capabilities like JavaScript or Python.

Development Tools

Development Tools are special programs that help programmers write other programs more easily and efficiently.

  • IDE/Code Editor : Where you write code (VS Code, PyCharm)

  • Compiler/Interpreter : Converts your code to machine language

  • Version Control : Git for tracking code changes

Career Paths in Programming

A series of jobs and roles that a developer can advance through as they gain more skills and experience.

  • Web Developer  : Build websites and web apps

  • Mobile Developer : Create phone/tablet apps

  • Data Scientist : Analyze data and create insights

  • Game Developer : Design and build games

  • DevOps Engineer : Manage software deployment

Summary

  • Programming creates software through code
  • Software types: system & application
  • Requires syntax & algorithms knowledge
  • Stages: design, code, test, deploy, maintain
  • Languages specialize in different domains
