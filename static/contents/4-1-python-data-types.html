Python Data Types

1. Numeric Types:

  1. int :

    Whole numbers (positive or negative), no decimals

    Example : 5, -3, 1000

  2. float :

    Numbers with decimal points

    Example : 3.14, -0.5, 2.0

  3. complex :

    Numbers with real and imaginary parts

    Example : 3+4j, 2j

# example of python numeric datatypes
integer_var = 10
float_var = 3.14
complex_var = 1 + 2j

print("Interger Numer : ", integer_var)
print("Float Numer : ", float_var)
print("Complex Number : ", complex_var)

 Output
                   Interger Numer :  10
Float Numer :  3.14
Complex Number :  (1+2j)

2. Sequence Types:

  1. str :

    Strings, immutable sequences of characters

    Example : "hello", 'Python'

  2. list :

    Lists, mutable ordered sequences of items

    Example : [1, 2, 3], ['apple', 'banana']

  3. tuple :

    Numbers with real and imaginary parts

    Example : (1, 2, 3), ('red', 'green')

  4. range :

    Represents an immutable sequence of numbers, often used in loops

    Example : range(5)

# example of python sequence datatypes
string_var = "Hello"
list_var = [1, 2, 3]
tuple_var = (4, 5, 6)
range_var = range(5)

print("String DataType : ", string_var)
print("List DataType : ", list_var)
print("Tuple DataType : ", tuple_var)
print("Range DataType : ", range_var)

 Output
                   String DataType :  Hello
List DataType :  [1, 2, 3]
Tuple DataType :  (4, 5, 6)
Range DataType :  range(0, 5)

3. Mapping Type:

  1. dict :

    Dictionaries, mutable collections of key-value pairs

    Example : {'name': 'Alice', 'age': 30}, {}

# example of python dict datatype
dict_var = {
    'name': 'John',
    'age': 25
}

print("Dictionary DataType : ", dict_var)

 Output
                   Dictionary DataType :  {'name': 'John', 'age': 25}

4. Set Type:

  1. set :

    Sets, mutable unordered collections of unique items

    Example : {1, 2, 3}, {'a', 'b', 'c'}

# example of python set datatype
set_var = {1, 2, 3, 3}

print("Set DataType : ", set_var)

 Output

Set DataType :  {1, 2, 3}

5. Boolean Type:

  1. bool :

    Boolean values, representing True or False

    Example : True, False

# example of python none datatype
bool_var = True

bool_var2 = False

print("Bool DataType : ", bool_var)
print("Bool DataType : ", bool_var2)


 Output

Bool DataType :  True
Bool DataType :  False

6. Binary Types:

  1. bytes :

    Immutable sequences of bytes.

    Example : b'hello'

  2. bytearray :

    Mutable sequence of bytes

    Example : bytearray(b'hello')

  3. memoryview :

    Provides a memory-efficient way to access the internal data of an object without copying it.

    Example : memoryview(b'hello')

7. None Type:

  1. NoneType :

    Represents the absence of value

    Example : None

# example of python bool datatype
none_var = None

print("None DataType : ", none_var)

 Output

None DataType :  None

Summary

  • Python has several built-in data types to represent different kinds of values.
  • Numeric types — int, float, complex — are used for mathematical operations.
  • Sequence types — str, list, tuple, range — store ordered collections of data.
  • Mapping type — dict — holds data as key-value pairs.
  • Set type — set — stores unique, unordered elements.
  • Boolean type — bool — represents truth values: True or False.
  • Binary types — bytes, bytearray, memoryview — handle byte data.
  • NoneType — represents the absence of a value using None.
