Python Identifiers & Variables

Identifiers

An identifier is a name given to programming elements like variables, functions, classes, etc. It helps in uniquely identifying these elements in the code.

Rules for Identifiers:

  • Can include letters (A-Z,a-z), digits (0-9), and underscores (_).

  • Must not start with a digit, e.g., 1nameis invalid.

  • Cannot be a keyword (reserved word like if,  for,  int).

  • Are case-sensitive, e.g., ageand Ageare different.

Example
# example of python identifiers
name = "Alice"      # Valid
_age = 25           # Valid
totalAmount = 100   # Valid
1stPlace = "John"   # Invalid (starts with a digit)
class = "Python"    # Invalid (uses a keyword)

Keywords

Keywords (also called reserved words) are special words in Python that have predefined meanings and functionalities. They are part of Python's syntax and cannot be used as identifiers
Python has 39 keywords (as of Python 3.12).

List of keywords:

  1. Value Keywords : True, False, None

  2. Operator Keywords : and, or, not, is, in

  3. Value Keywords : True, False, None

  4. Control Flow Keywords : if, elif, else for while break continue pass try except fianlly raise assert

  5. Function and Class : def, return, lambda yield class

  6. Context Management : with, as,

  7. Import and Module : import, from,

  8. Scope and Namespace : global, nonlocal,

  9. Async Programming : async, await,

Variables

A variable is a named storage location in memory that holds a value. It allows data to be stored, modified, and retrieved in a program.

Key Properties:

  • Must have a unique identifier (name)..

  • Has a data type (e.g. int , float, str).

  • Can be reassigned (in most languages).

Example
# example of python variables
age = 30           # 'age' is the variable storing value 30
name = "Bob"       # 'name' stores a string
price = 45.99      # 'price' stores a float

Variables vs Keywords

Variables Keywords
Used to store data. Reserved words with predefined meanings in Python.
Can be created, modified, and deleted by the programmer. Cannot be modified or used as variable names.
Examples: x, age, name Examples: if, while, for
Hold values that are manipulated in the program. Used to define the structure of Python code.
Variable names must follow naming rules but are otherwise flexible. Fixed by Python language and cannot be altered.

Summary

  • Identifiers are names for variables, functions, and classes that follow naming rules.
  • Keywords are reserved words in Python and cannot be used as identifiers.
  • Variables store data values and can be changed during execution.
