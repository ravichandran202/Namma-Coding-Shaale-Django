Common Dictionary Errors

Learn about common dictionary errors and how to fix them

1. KeyError - Accessing Non-existent Key

Problem: Trying to access a key that doesn't exist 
# This will cause ERROR
student = {"name": "Ram"}
print(student["age"])   # KeyError: 'age'
 Error
KeyError: 'age'
# Safe ways to access
student = {"name": "Ram"}

# Method 1: Use get() with default
print(student.get("age", "Not Found"))  # Not Found

# Method 2: Check with 'in' first
if "age" in student:
    print(student["age"])
else:
    print("Age not found")

# Method 3: Use setdefault()
age = student.setdefault("age", 22)
print("Age:", age)
print("Student:", student)
 Output
Not Found
Age not found
Age: 22
Student: {'name': 'Ram', 'age': 22}

2. TypeError - Unhashable Type

Problem: Using mutable objects as dictionary keys 
# These will cause ERROR
# data = {["a", "b"]: 1}        # TypeError: unhashable type: 'list'
# data = {{"a": 1}: "value"}    # TypeError: unhashable type: 'dict'
# data = {{1, 2}: "value"}      # TypeError: unhashable type: 'set'
# Correct - use immutable keys
# Strings as keys
data1 = {"name": "Ram", "city": "Bangalore"}
print("String keys:", data1)

# Numbers as keys
data2 = {1: "one", 2: "two", 3.5: "three point five"}
print("Number keys:", data2)

# Tuples as keys (immutable)
data3 = {("Ram", "CS"): "Student", ("Bangalore", "India"): "Location"}
print("Tuple keys:", data3)

# Boolean as keys
data4 = {True: "Yes", False: "No"}
print("Boolean keys:", data4)
 Output
String keys: {'name': 'Ram', 'city': 'Bangalore'}
Number keys: {1: 'one', 2: 'two', 3.5: 'three point five'}
Tuple keys: {('Ram', 'CS'): 'Student', ('Bangalore', 'India'): 'Location'}
Boolean keys: {True: 'Yes', False: 'No'}

3. KeyError with pop()

Problem: pop() raises KeyError for non-existent keys 
# This will cause ERROR
student = {"name": "Ram"}
# student.pop("age")   # KeyError: 'age'
# Safe ways to use pop()
student = {"name": "Ram", "age": 22}

# Method 1: Use default value
marks = student.pop("marks", "Not found")
print("Marks:", marks)
print("Student:", student)

# Method 2: Check before popping
if "age" in student:
    age = student.pop("age")
    print("Removed age:", age)
else:
    print("Age not found")

print("Final student:", student)
 Output
Marks: Not found
Student: {'name': 'Ram', 'age': 22}
Removed age: 22
Final student: {'name': 'Ram'}

5. AttributeError - Wrong Method

Problem: Using list methods on dictionaries 
# These will cause ERROR
student = {"name": "Ram"}
# student.append("age")        # AttributeError: no append
# student.insert("age", 22)    # AttributeError: no insert
# student.sort()               # AttributeError: no sort
# Correct dictionary methods
student = {"name": "Ram"}

# Add elements
student["age"] = 22                    # Use assignment
student.update({"city": "Bangalore"})  # Use update()
print("After adding:", student)

# Remove elements
student.pop("age")                     # Use pop()
print("After pop:", student)

# Get all elements
print("Keys:", list(student.keys()))
print("Values:", list(student.values()))
print("Items:", list(student.items()))
 Output
After adding: {'name': 'Ram', 'age': 22, 'city': 'Bangalore'}
After pop: {'name': 'Ram', 'city': 'Bangalore'}
Keys: ['name', 'city']
Values: ['Ram', 'Bangalore']
Items: [('name', 'Ram'), ('city', 'Bangalore')]
Dictionary Errors Summary
Error Cause Solution
KeyError Accessing non-existent key Use get() or check with in
TypeError Mutable key (list, dict, set) Use immutable keys (str, int, tuple)
KeyError with pop pop() on non-existent key Use default value: pop(key, default)
RuntimeError Modifying dict during iteration Iterate over list(dict.keys())
AttributeError Wrong method (append, sort, etc.) Use dict methods (update, pop, get)
MemoryError Extremely large dictionary Process data in chunks

Try It Yourself

Fix the errors in the following code:

  1. student = {"name": "Ram"}; print(student["age"])
  2. data = {["a", "b"]: 1}
  3. d = {"a": 1}; d.pop("b")
  4. d = {"a": 1, "b": 2}; for k in d: d["c"] = 3
  5. d = {"a": 1}; d.append("b")
# 1. Fix KeyError - use get() or check first
student = {"name": "Ram"}
print("1. Age:", student.get("age", "Not found"))

# 2. Fix TypeError - use tuple instead of list
# Wrong: data = {["a", "b"]: 1}
# Correct:
data = {("a", "b"): 1}
print("2. Fixed:", data)

# 3. Fix KeyError in pop - use default value
d = {"a": 1}
value = d.pop("b", "Key not found")
print("3. Pop result:", value)

# 4. Fix RuntimeError - iterate over copy
d = {"a": 1, "b": 2}
for k in list(d.keys()):  # Create list copy
    d["c"] = 3
print("4. After safe iteration:", d)

# 5. Fix AttributeError - use correct method
d = {"a": 1}
# Wrong: d.append("b")
# Correct:
d["b"] = 2  # Use assignment
print("5. After adding:", d)
 Output
1. Age: Not found
2. Fixed: {('a', 'b'): 1}
3. Pop result: Key not found
4. After safe iteration: {'a': 1, 'b': 2, 'c': 3}
5. After adding: {'a': 1, 'b': 2}
Remember
  • Always use get() or check with in before accessing keys
  • Dictionary keys must be immutable (strings, numbers, tuples)
  • Use pop(key, default) to avoid KeyError
  • Never modify dictionary while iterating - use list(dict.keys())
  • Use proper dictionary methods, not list methods
  • For large data, process in chunks to avoid MemoryError
