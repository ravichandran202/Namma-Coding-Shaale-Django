Method Reference
list.append(x)
Adds an item x to the end of the list.
list.insert(i, x)
Inserts item x at a given position i.
list.pop([i])
Removes and returns item at position i. Default is last item.
list.remove(x)
Removes the first item from the list whose value is equal to x.
list.extend(iterable)
Extends list by appending all items from the iterable (e.g., another list).
list.sort()
Sorts the items of the list in place (modifies original).
